Default Fit Function & Algorithm

The plugin tries to find a normal diffusion model and Levenberg-Marquardt fit routine.

Fit Parameter Presets

This plugin reads default values for the parameters either from an internal declaration of the fit function, or from a fitparams_fcs_fit.ini file which is either situated in the global configuration directory, or in the suer's configuration directory. THe user file superseeds the global file.

These configuration files have a specific format (note: none of the information in the file and not even their existence is obligatory!): 


[FITFUNCTION_ID1]
PARAMETER_ID1 = (number): default value of this parameter
PARAMETER_ID1_fix = (true|false): default fix-state of this parameter
PARAMETER_ID1_max = (true|false): default lower-bound of this parameter
PARAMETER_ID1_min = (number): default lower-bound of this parameter
PARAMETER_ID2 = (number): default value of this parameter
PARAMETER_ID2_fix = (true|false): default fix-state of this parameter
PARAMETER_ID2_max = (true|false): default lower-bound of this parameter
PARAMETER_ID2_min = (number): default lower-bound of this parameter

[FITFUNCTION_ID2]
PARAMETER_ID1 = (number): default value of this parameter
PARAMETER_ID1_fix = (true|false): default fix-state of this parameter
PARAMETER_ID1_max = (true|false): default lower-bound of this parameter
PARAMETER_ID1_min = (number): default lower-bound of this parameter
PARAMETER_ID2 = (number): default value of this parameter
PARAMETER_ID2_fix = (true|false): default fix-state of this parameter
PARAMETER_ID2_max = (true|false): default lower-bound of this parameter
PARAMETER_ID2_min = (number): default lower-bound of this parameter
Here FITFUNCTION_ID2 is the ID of the respective fit-function (see e.g. the list of implemented ids: under Help|About Plugins in the main window, here, or the respective fit function's plugin documentation. The PARAMETER_IDn stand for the different parameters of the fit functions (see the fit function documentation).
