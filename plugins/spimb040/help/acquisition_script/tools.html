Logging Functionsl

  • logText(data1[, data2, ...]) - write normal text to the logging pane:
    you may specify several parameters that will all be converted to strings and written out one after the other.
  • logWarning(data1[, data2, ...]) - write a warning to the logging pane:
    you may specify several parameters that will all be converted to strings and written out one after the other.
  • logError(data1[, data2, ...]) - write an error messaget to the logging pane:
    you may specify several parameters that will all be converted to strings and written out one after the other.
