Status Report Functions

  • tools.setStatus(message) - write text (ncluding HTML markup!) to the status line below the script editor:
  • tools.setStatusProgressRange(minimum, maximum) - activates the progress bar in the status spin and sets its minimum and maximum values as given (floating point numbers):
    the actual progress is set, using tools.setStatusProgress(value) or tools.incStatusProgress().
  • tools.setStatusProgress(value) - sets the current progress in the status spin to the specified value (floating point numbers):
    The range of values can be specified with tools.setStatusProgressRange(minimum, maximum).
  • tools.incStatusProgress() - increments the current progress in the status spin by 1:
    The range of values can be specified with tools.setStatusProgressRange(minimum, maximum).
