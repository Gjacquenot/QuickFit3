Illumination Control

  • instrument.setLaserIntensity(laser, line, intensity) - sets the laser intensity :
    This function sets the intensity of the specified laser (either 1 or 2). Each laser may have several lines (the first one is 0) and you may set the power in device units (usually milliwats) as a floating point value.
  • instrument.setBrightfieldIntensity(lightsource, line, intensity) - sets the transmission illumination intensity :
    This function sets the intensity of the transmission illumination. The illumination device may have several lines (the first one is 0) and you may set the power in device units as a floating point value.
    Currently this function does NOT block, so use sleepMS() to wait a sufficient time until the new intensity is settled.

Shutter Control

  • instrument.setShutter("shutter", state) - sets the state (boolean) of the specified shutter :
    These shutters are defined:
    • "main": the main shutter
    • "laser1": the laser 1 shutter
    • "laser1": the laser 2 shutter
    • "transmission": the transmission illumination shutter
    This function blocks until the shutter has reached its final position.

Filter Changer Control

  • instrument.setFilterWheel("wheel", filter) - sets the filter (integer) of the specified wheel :
    These wheels are defined:
    • "detection": the detection arm filter wheel
    This function blocks until the wheel has reached its final position.

Stage Control

  • instrument.moveStageRel("stage", delta) - move the specified stage by a given distance delta :
    These stages are defined:
    • "x": sample translation stage: x-direction
    • "y": sample translation stage: y-direction
    • "z": sample translation stage: z-direction
    The new stage position is a displacement relative to the current position (current_position + delta) and has to be given in micrometers. This function blocks until the stage stops moving.
  • instrument.moveStageAbs("stage", new_position) - move the specified stage to the given position :
    These stages are defined:
    • "x": sample translation stage: x-direction
    • "y": sample translation stage: y-direction
    • "z": sample translation stage: z-direction
    The stage position is an absolute value and has to be given in micrometers. This function blocks until the stage stops moving.
