Illumination Control

  • setLaserIntensity(laser, line, intensity) - sets the laser intensity :
    This function sets the intensity of the specified laser (either 1 or 2). Each laser may have several lines (the first one is 0) and you may set the power in device units (usually milliwats) as a floating point value.
  • setTransmissionIntensity(line, intensity) - sets the transmission illumination intensity :
    This function sets the intensity of the transmission illumination. The illumination device may have several lines (the first one is 0) and you may set the power in device units as a floating point value.

Shutter Control

  • setShutter(shutter, state) - sets the state (boolean) of the specified shutter :
    These shutters are defined:
    • "main": the main shutter
    • "laser1": the laser 1 shutter
    • "laser1": the laser 2 shutter
    • "transmission": the transmission illumination shutter
