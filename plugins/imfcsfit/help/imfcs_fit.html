This plugin allows to perform fits for imaging FCS data. It shares most functionality with the FCS Curve Fitting Plugin. There is only one major difference between these two plugins: The FCS fit plugin stores the fit results for each run in a separate column/row/set of fit results, whereas this plugin creates one set for the average run and one set for all other runs. The fit results are then stored as vectors/matrices inside this set. This saves a considerable amount of memory in the project file and while QuickFit is running. Also it improves the speed of execution of the imFCS raw data record Plugin.
This plugin allows to perform fits on several processor cores in parallel (multi-threaded). Set the maximum number of processors to use in the general settings dialog.

  1. imFCS Curve Fit User Interface
  2. Explanation of Fit Statistics
  3. Configuration (global fit parameter presets ...)
  4. Configuration options in the mains ettings dialog
  5. Hints for fit model programmers

This plugin uses any of the available Fit Algorithm Plugins to execute the model fit: $$list:fitalg:$$

Models have to be implemented as Fit Function Plugins. Models that may be used with this plugin are: $$list:fitfunc:fcs_$$ $$list:fitfunc:fcds_$$ $$list:fitfunc:dls_$$
