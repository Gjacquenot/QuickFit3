This plugin provides importer plugins for PicoQuant TCSPC computer hardware.

  • Importer for PicoQuant TTTR (*.t3r) (e.g. TimeHarp computer cards)
  • Importer for PicoQuant PicoHarp 300 TTTR (*.pt3)
  • Importer for PicoQuant Unified TTTR File (*.ptu)
