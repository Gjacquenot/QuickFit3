Lifetime Preview

This dialog allows the get a quick peek at the microtime histogram that can be extracted from the TCSPC input data. From this histogram, the fluorescence lifetimes of the different species in the measurement can be extracted. Also this dialog can be used to easily set the ranges for the lifetime filter for FCS/FCCS evaluation.

You can open this dialog by clicking the "Peek Lifetime-Distribution" button in the Input row of the TCSPC Importer Dialog. The tool will then read the start of your specified TCSPC file (i.e. either the first 50,000 photons or the first 10s, whichever happens first) and build a lifetime histogram from the microtime data stored in that file. Then the normalized histogram is displayed in the following dialog:

In addition to haveing a look at the microtime distribution, you can also use this dialog to define the lifetime ranges for the lifetime filter for FCS/FCCS evaluation:

  1. activate the lifetime-range mode by checking the button at the top-left.
  2. Then you can simply "draw" a range rectangle int the graph (keeping the left mouse-button pressed down, while dragging the mouse ofver the plot). The newly selected range will be displayed in the plot.
  3. If you are satisfied, you can right-click the plot and select one of the use lifetime range for channel X menu entries to take the range over into the TCSPC Importer Dialog.
