This tool will calculate histograms of the microtime (i.e. the picosecond-resolution time-tag part) in the TCSPC records. The results will be imported into the QuickFit project as table RDRs, which contain the histogram data and plots for each channel. YYou can choose to cut the input file into several segments. Then a histogram for each channel in each segment is calculated ans displayed.
