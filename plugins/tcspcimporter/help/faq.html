$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ $$faq_start$$ Ho can I perform FCS/FCCS data evaluation, using TCSPC data files acquired on a microscope? $$faq_answer$$ QuickFit contains the tcspcimporter plugin, which allows to read and process TCSPC data from a number of diferent input file formats. It offers an option to directly correlate the TCSPC data with different algorithms. Finally the FCS/FCCS data evaluation can be perfromed with diferent othe rplugins, such as FCS Curve Fitting, FCS MaxEnt evaluation, an FCS MSD Evaluation, an FCCS global fit etc.
