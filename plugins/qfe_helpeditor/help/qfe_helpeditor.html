$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This extension allows to edit (with preview) QuickFit 3 online-help pages.
  • You can load an existing help page, or create a new one, also based on predefined templates.
  • The QuickFit online-help is based on HTML documents, that are enriched with special "tags", which are interpreted by QuickFit. These tags are enclosed in $$...$$ and can represent different features:
    • $$math:LATEX$$ and $$bmath:LATEX$$ will render the enclosed LaTeX expression as an image and include it. This can be used for easy expression rendering
    • $$note:TEXT$$ will create a box with TEXT contained
    • $$ref:ID:text$$ defines a literature reference, which can be reused later using $$ref:ID:$$. All references will be summarized in a list at the end of the page, where $$references$$ is inserted into the document.
    • Several short forms for different types of information are provided ...
    • A complete list of possible help tags can be found in the help system reference.

    All these markups can be inserted with the buttons provided above the edit widget for the help file, or via the context-menu (activated by a right-click) of the edit widget.
  • A click on Preview at the bottom of the window will open a preview of the current help page.
  • You can insert images, or even paste them from the clipboard, using .
