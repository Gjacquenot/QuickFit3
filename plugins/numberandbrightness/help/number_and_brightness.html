Functionality

This plugin calculates the particle number and particle brightness from a set of images according to the well-established N&B-method. You can find a detailed description and discussion of this method in these papers:
  • Hong Qian, Elliot L. Elson (1990): "Distribution of molecular aggregation by analysis of fluctuation moments", PNAS, 87(14), 5479-5483, Pubmed: 2371284
  • Jay R. Unruh, Enrico Gratton (2008): "Analysis of Molecular Concentration and Brightness from Fluorescence Fluctuation Data with an Electron Multiplied CCD Camera", Biophysical Journal, 95(11), 5385–5398, Pubmed: 18805922
According to these references, four images are needed:
  1. an intensity image $$math:I$$, best taken with an exposure time smaller than the particle diffusion time.
  2. an intensity variance image $$math:\sigma_I^2$$, e.g. calculated from a series of images.
  3. a background intensity image $$math:B$$, taken with the same settings as image $$math:I$$, but without illumination.
  4. a background variance $$math:\sigma_B^2$$
From these images for every pixel, the plugin calculates the particle number $$math:N$$ and the particle brightness $$math:\epsilon$$: $$bmath:N=\frac{(I-B)^2}{\sigma_I^2-\sigma_B^2}$$ $$bmath:\epsilon=\frac{\sigma_I^2-\sigma_B^2}{I-B}$$ Note however that these quantities resemble the true values ONLY if the images were taken with a photon counting detector, as e.g. in a confocal laser scanning microscope equipped with APDs. If the detector is an analog detector (e.g. an CCD- or CMOS-camera), additional noise due to the analog-to-digital conversion is generated and the quantities above do no longer resemble the true values. A detailed discussion of this topic is given in the second reference by Unruh and Elson. $$qf_commondoc_backtop$$


Usage/User Interface

This plugin adds a number-and-brightness raw data record type to QuickFit, which calculates the desired values on the fly. Several options are supplied to add such a record to the project.

number and brightness display editor

The number and brightness tab of the raw data record is split into two tabs with plots:
  1. image plots: This tab shows the three important images: particle number $$math:N$$, particle brightness $$math:\epsilon$$ and the intensity image $$math:I$$. The fourth plot shows a scatter plot of the particle number against the particle brightness.
  2. histograms: Shows histograms of the particle number and particle brightness images. Next to the histograms you see a summary of the statistics of these images. Note: Histograms of the input images can be found in the "overview images" tab of the raw data record editor!
