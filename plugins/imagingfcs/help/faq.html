$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ $$faq_start$$ What is an imaging fluorescence correlatioon spectroscopy (imFCS)? $$faq_answer$$ In standard FCS, measurements are performed for single positions in a sample. In imFCS, an array detector (fast camera) is used to record the fluorescence fluctuations from several pixels in the sample simultaneously. The for each pixel in the measurement, an autocorrelation (and possibly pixel-pixel or two-color-cross-correlation) curves are calculated and evaluated. Finally the fit parameters (e.g. the diffusion coefficient) can be represented as a map or image for the sample, reporting on how fast particles move at different positions in the sample. imFCS is usually impelemnted on a total internal reflection fluorescence microscope (TIRF) or a single plane illumination microscope (SPIM). $$faq_end$$ $$faq_start$$ How can I correlate an image series in QuickFit for an imFCS data evaluation? $$faq_answer$$ The help page Correlating imFCS data and the tutorials imFCS analysis of beads in solution and imFCS analysis in a cell give a step-by-step introduction to this task. $$faq_end$$ $$faq_start$$ Can I simulate an imFCS/imFCCS experiment? $$faq_answer$$ Yes, a simple simulator is implemented in QuickFit. It creates an image stack from the random walk of several fluorescenctly labeled particles in a plane. See imFCS Simulation for details. $$faq_end$$ $$faq_start$$ How are imFCCS data represented in QuickFit? $$faq_answer$$ In imFCCS experiments, several auto- and cross-correlation curves exist for every pixel in the measurement (e.g. two auto- [ACF0,ACF1] and one two-color cross-correlation [CCF/DCCF/FCCS] curves). QuickFit generates one raw data record (RDR) for each of these curves (all in the same RDR group) and an imFCCS global fit may recognize these several records and fit them all together.
