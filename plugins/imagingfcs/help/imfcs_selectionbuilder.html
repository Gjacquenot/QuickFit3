Imaging FCS Plugin: Selection Builder

This dialog allows to select/mask pixels by a given set of rules.

A pixel is selected/masked only, if all actived conditions to the selected fit parameters and/or overview images are met. Conditions may be:

  • in-range: the parameter/image value is between two given values (i.e. >= the left boundary and <= the right boundary)
  • out-of-range: the parameter/image value is outside two given values (i.e. < the left boundary and > the right boundary)
  • above threshold: the parameter/image value is above a given threshold value
  • below threshold: the parameter/image value is below a given threshold value
Also it is possible to exclude masked pixels from the selection.

After a click on "OK", all pixels, matching the given conditions are beeing selected.
