Introduction

This plugin reads image correlation data and allows to evaluate it in QuickFit $$version$$ (model fits, ...). Once imported the data looks to QuickFit like usual FCS correlation curves, so you can use all evaluation modules available for these. In addition to giving access to the correlation curves the raw data record editor allows to draw different maps of any evaluation result. Here they can also be exported as images or data arrays for post-processing in other computer programs.
