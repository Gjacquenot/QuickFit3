This plugin uses the Cairo graphics library to create PDF, EPS, PS and SVG files from any plot in QuickFit.

This plugin currently has some limitations:

  • text is not rendered as editable text, but as paths. So if you look at a created file, the display is good, but if you loadit in a vector graphics program, you won't be able to change the text.
  • no fill patterns are supported at the moment
