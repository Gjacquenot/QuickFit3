This plugin uses the Cairo graphics library to create PDF, EPS, PS and SVG files from any plot in QuickFit.

This plugin currently has some limitations:

  • There are two options for text rendering: The exporters with the name addition "[editable text]" export text in a way, that it is editable in graphics software afterwards. the exporters without this addition render text as polygons/paths, so the text is no longer editable in a vector graphics editing program, but the rendering can be more accurate!
  • no fill patterns (only colors) are supported at the moment
