$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin implements a fit function, which only contains exponential decay terms: $$bmath:G(\tau)=G_\infty+a_1\cdot\exp\left(-\frac{\tau}{\tau_1}\right)+a_2\cdot\exp\left(-\frac{\tau}{\tau_2}\right)$$ The parameters are:
  • $$math:G_\infty$$: offset of the correlation function
  • $$math:a_1, a_2$$: exponential term amplitudes
  • $$math:\tau_1,\tau_2$$: exponential decay times
