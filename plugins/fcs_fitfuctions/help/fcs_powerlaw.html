$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin implements a fit function, which only contains power law decay terms: $$bmath:G(\tau)=G_\infty+a_1\cdot\tau^{-b_1}+a_2\cdot\tau^{-b_2}$$ The parameters are:
  • $$math:G_\infty$$: offset of the correlation function
  • $$math:a_1, a_2$$: amplitudes
  • $$math:b_1,b_2$$: powers
