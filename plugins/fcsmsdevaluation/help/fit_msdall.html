This dialog can be used to fit a power law $$bmath:\langle r^2(\tau)\rangle=P\cdot D\cdot\tau^\alpha$$ ($$math:P=6$$ for 3D-diffusion, $$math:P=4$$ for 2D-diffusion, $$math:P=2$$ for 1D-diffusion) to a selected range of MSD curves (e.g. the current only, or all runs of the current file, ...). Simply specify the range in which to fit, the initial values and the type of fit to perform and click on "Fit". All files and/or runs, selected in "apply to" will be fitted to the actvated models. The fit results will be stored in the "theory curves" of the runs/files.

Several fit methods are available (although all fit the same model function):

  1. Levenberg-Marquardt-Fit (LM-Fit): This method uses the standard LM least squares fit routine to fit the power law model
  2. regression: This uses a simple regression to fit a line to the double-logarithmically plotted data
  3. robust regression (IRLS): This uses a robust (iteratively reweighted least squares) regression to fit a line to the double-logarithmically plotted data. See http://cnx.org/content/m45285/latest/ for further information on IRLS.
