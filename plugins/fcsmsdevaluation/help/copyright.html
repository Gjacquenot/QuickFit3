This plugin uses the Levenberg-Marquardt Least-Squares scheme implemented in the library lmfit:

Joachim Wuttke: lmfit - a C/C++ routine for Levenberg-Marquardt minimization with wrapper for least-squares curve fitting, based on work by B. S. Garbow, K. E. Hillstrom, J. J. Moré, and S. Moshier. Version 5 from http://apps.jcns.fz-juelich.de/doku/sc/lmfit.
