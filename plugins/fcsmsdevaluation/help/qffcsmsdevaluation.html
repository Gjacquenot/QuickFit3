Introduction

Ths plugn implements the maximum entropy (MaxEnt) data fit method for fluorescence correlation spectroscopy (FCS) or dynamic lightscattering (DLS) data, as described in these references:
  • R.K. Bryan (1990): "Maximum entropy analysis of oversampled data problems", European Biophysics Journal 18(3), 65-174, DOI: 10.1007/BF02427376
  • Károly Módos, Rita Galántai, Irén Bárdos-Nagy, Malte Wachsmuth, Katalin Tóth, Judit Fidy und Jörg Langowski (2004): "Maximum-entropy decomposition of fluorescence correlation spectroscopy data: application to liposome–human serum albumin association", European Biophysics Journal 33(1), 59-67, DOI: 10.1007/s00249-003-0343-6

Supported Models

This plugin supports several models that incorporate a MaxEnt distribution $$math:(\tau_i, p(\tau_i)=p_i)_{i=1...N}$$:
  1. FCS: 3D diffusion with Triplet: This plugin may be used for FCS data from a confocal microscope with assumed 3D diffusion. The model is: $$bmath:g(\tau)=\left(\frac{1-\Theta_T+\Theta_T\cdot\exp(-tau/\tau_T)}{1-\Theta_T}\right)\cdot\sum_{i=1}^Np_i\cdot\left(1+\frac{\tau}{\tau_i}\right)^{-1}\cdot\left(1+\frac{\tau}{\gamma^2\cdot\tau_i}\right)^{-1/2}$$ Here $$math:\Theta_T$$ is the triplet fraction and $$math:\tau_T$$ is the triplet lifetime. The parameter $$math:\gamma$$ describes the axial ratio of the gaussian laser focus (a common value for confocal microscopes is $$math:\gamma\approx 6$$).
