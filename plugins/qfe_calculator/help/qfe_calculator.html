This plugin implements a simple math expression parser. It can be used to perform quick calculations directly inside QuickFit.

The expression parser supprts these functions and variables:

    $$insertglobal:parseref.inc$$
  • variables containing former results
  • print/output functions
  • Plotting Functions
    • leneral plotting functions
    • logarithmic plots
    • plots with errorbars
    • scatter
    • bar charts/histograms
