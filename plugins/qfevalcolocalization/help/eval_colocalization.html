Introduction

This plugin allows to perform a simple colocalization analysis on a dual-color image stack. It plots the pixel intensitiess into a 2D-correlation plot and calculates correlation coeffciients between the channels. Before any data evaluation, the plugin subtracts a background intensity from each pixel and filters pixels, which are below a given intensity threshold (typically 0, so not used!). See $$ref:DUNN2011:Dunn, K. W., Kamocka, M. M., & McDonald, J. H. (2011). A practical guide to evaluating colocalization in biological microscopy. American Journal of Physiology - Cell Physiology, 300(4), C723–C742. doi:10.1152/ajpcell.00462.2010$$ for an introduction to colocalization in biology.

In addition to creating a correlation plot between two color chanels, this plugin also calculates different coefficients of colocalization between the two color channel $(G_I)$ and $(R_i)$

  • Pearson's correlation coefficient (PCC): $[\rho_{G,R}=\frac{\sum_i\left(G_i-\overline{G}\right)\cdot\left(R_i-\overline{R}\right)}{\sqrt{\left[\sum_i\left(G_i-\overline{G}\right)^2\right]\cdot\left[\sum_i\left(R_i-\overline{R}\right)^2\right]}},]$ where $(\overline{G},\overline{R})$ are the averages over all $(N)$ pixels $(G_i, R_i)$: $[ \overline{G}=\frac{1}{N}\cdot\sum_{i=1}^NG_i ]$
  • Manders Overlap Coefficient (MOC): $[\mbox{MOC}_{G,R}=\frac{\sum_i\left(G_i\cdot R_i\right)}{\sqrt{\left[\sum_iG_i^2\right]\cdot\left[\sum_iR_i^2\right]}}]$

User Interface

  • Choose an image stack from the list on the right
  • select the channels, that you want to compare
  • observe the results on the left
  • You can draw a rectangle in the correlation plot with the mouse. When that rectangle is finished, all pixels with values in that parameter region will be highlighted (in blue)n in the overview image.
  • This plugin also calculates Pearson's correlation coefficient between the two color channels for all pixels larger than the background threshold and (optionally) within the user-selected range in the correlation plot. These results are stored as evaluation results (last tab) and displayed in the table, next to the overview plot.
  • The button "estimate background" estaimtes the background intensity as the lower 5% quartile of each color channel.
