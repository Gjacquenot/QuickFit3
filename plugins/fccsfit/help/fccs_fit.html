This plugin allows to perform fits for FCCS data.
This plugin allows to perform fits on several processor cores in parallel (multi-threaded). Set the maximum number of processors to use in the general settings dialog.

  1. FCCS Curve Fit User Interface
  2. Fit Weighting
  3. Explanation of Fit Statistics
  4. Configuration (global fit parameter presets ...)
  5. Configuration options in the mains ettings dialog
  6. Hints for fit model programmers

This plugin uses any of the available Fit Algorithm Plugins to execute the model fit: $$list:fitalg:$$

Models have to be implemented as Fit Function Plugins. Models that may be used with this plugin are: $$list:fitfunc:fcs_$$ $$list:fitfunc:fccs_$$ $$list:fitfunc:dls_$$
