$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin uses a subset of the available QuickFit Fit models. When implementing a model for the imaging FCCS Fit plugin, please keep these hints in mind:
  • Some parameter names have special meaning and QuickFit will possibly fill in a special value:
    • count_rate is the countrate measured together with the FCS curve (if available) in Hz or counts per second
    • count_rate1 is the countrate measured together with the FCCS curve in channel 1 (if available) in Hz or counts per second
    • count_rate2 is the countrate measured together with the FCCS curve in channel 2 (if available) in Hz or counts per second
    • pixel_width, pixel_height are the width and height of image pixels (usually also taking care of binning) in nanometers.
    • dls_angle the scattering angle for e.g. dynamic light scattering in degrees.
    • wavelength the wavelength for e.g. dynamic light scattering in nanometers.
    • refractive_index the refractive indes for e.g. dynamic light scattering.
    • refractive_index the sample viscosity for e.g. dynamic light scattering centi Poise (cp).
    • temperature the temperature for e.g. dynamic light scattering in Kelvin (K).
    • focus_distance the distance of two foci in 2fFCS/DCCF in nanometers.
    • focus_distancex the distance of two foci in x-direction in a 2fFCS/DCCFmeasurement in nanometers.
    • focus_distancey the distance of two foci in y-direction in a 2fFCS/DCCFmeasurement in nanometers.
  • It is possible to overwrite the QFFitFunction given initial parameter values with the QFEvaluationItem's QFProperties interface, by setting a property of the following names to the desired initial value/err/fix:
    • PRESET_PARAMID: overwrites the initial values of the parameter PARAMID (in all upper-case letters).
    • PRESET_PARAMID_ERROR: overwrites the initial error of the parameter PARAMID (in all upper-case letters).
    • PRESET_PARAMID_FIX: overwrites the initial fix state of the parameter PARAMID (in all upper-case letters).
