Table Access Functions

  • data(row, column) - access a table cell (row, column):
    returns the value stored in table cell (row, column)
  • dataleft(delta_column) - return the value of a cell, left of the current cell:
    accesses the cell (row, column-delta_column), where (row, columns) is the current cell. So if you want to calculate the sine of the column to the left of the current column, use the expression sin(dataleft(1)). Note that you do NOT need a -1 to access a cell on the left. Use negative numbers to access cells on the right!
