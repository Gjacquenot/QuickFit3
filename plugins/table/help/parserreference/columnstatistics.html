Simple Column Statistics Functions

  • colavg(column, row_start, row_end) - average over rows row_start .. row_end (inclusive) in the given column
    colavg(column) - average over all rows in the given column
    calculates $$bmath:\overline{data}=\frac{1}{N}\cdot\sum_{i=row\_start}^{row\_end}data(column, i)$$
  • colsum(column, row_start, row_end) - sum over rows row_start .. row_end (inclusive) in the given column
    colsum(column) - sum over all rows in the given column
    calculates $$bmath:\sum_{i=row\_start}^{row\_end}data(column, i)$$
  • colsum2(column, row_start, row_end) - sum squares over rows row_start .. row_end (inclusive) in the given column
    colsum2(column) - sum squares over all rows in the given column
    calculates $$bmath:\sum_{i=row\_start}^{row\_end}data(column, i)^2$$
  • colvar(column, row_start, row_end) - variance over rows row_start .. row_end (inclusive) in the given column
    colvar(column) - variance over all rows in the given column
  • colstd(column, row_start, row_end) - standard deviation over rows row_start .. row_end (inclusive) in the given column
    colstd(column) - standard deviation over all rows in the given column
  • colmin(column, row_start, row_end) - minimum over rows row_start .. row_end (inclusive) in the given column
    colmin(column) - minimum over all rows in the given column
  • colmax(column, row_start, row_end) - maximum over rows row_start .. row_end (inclusive) in the given column
    colmax(column) - maximum over all rows in the given column
  • colmedian(column, row_start, row_end) - median over rows row_start .. row_end (inclusive) in the given column
    colmedian(column) - median over all rows in the given column
  • colquantile(column, row_start, row_end, quantile) - given quantile over rows row_start .. row_end (inclusive) in the given column
    colquantile(column, quantile) - given quantile over all rows in the given column
