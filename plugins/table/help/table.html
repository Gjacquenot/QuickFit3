Description

This plugin allows the user to view/edit a table of values. It implements these features:

  • The values may be of diferent datatypes (integer or floating point number, text, ...).
  • Data may be loaded from or saved to external files.
  • Data exchange with other applications (Excel, Origin, ...) and other QuickFit plugins is usually possible via the clipboard.
  • Many common table editing functions are implemented (insert/delete rows/cols, table resize, ...).
  • A expression parser allows to calculate the contents of cells in dependence of the contents of other cells.
  • It is possible to create plots of the data in the table. Each column is seen as a single dataseries for the plot.

Usage Notes & Hints

  • You may insert either an "Editable Table" or a "Table From File" into a QuickFit project. The lather one may NOT be edited by the user, as it simply represents the contents of a file. If you want to edit the contents of a file insert an "Editable Table" and then load the file using . The data of an Editable Table is stored inside the QuickFit project file, so such a raw data record does NOT depend on any external file.
