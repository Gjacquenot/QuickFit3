This plugin allows to read image series in the RAW format (64-bit unsigned integers), which simply contains the frames as binary data in row-major format and with separate channels, i.e. (frame1.ch1,rame1.ch2,...,frame2.ch1,frame2.ch2,...). An additional text file filename.description.txt has to be present, which is read to get the frame width/height and number of channels [default:1]. here is an example: 

width=width

height=height

channels=channels

bitspersample=64
