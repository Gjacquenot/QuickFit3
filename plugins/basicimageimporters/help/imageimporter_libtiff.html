This plugin allows to read image series from TIFF files. The plugin also recognizes and reads files with several color-channels. To open and read these it uses the open source library libtiff.

Limitations: libTIFF is able to read many different types of TIFF images, also many different types of compressions (e.g. JPEG, ZIP/defalte, LZW, ...), but it has one internal limitation: It is not possible to read more than around 60000 images from one single file.
