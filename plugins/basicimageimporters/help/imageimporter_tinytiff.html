Description of the plugin

This plugin allows to read image series from TIFF files. It uses a simple home-built implementation of the TIFF file format with only a limited set of supported features.

Limitations:

  • no compression of any type is supported
  • only 8, 16 and 32 bits per sample are supported
  • only planar TIFF files are supported by this plugin (i.e. pixels samples have to be organized as R R R ... R R R G G G ... G G G B B B ... B B B ...

TinyTIFF library

This plugin uses the TinyTIFF library, which was developed by Jan W. Krieger. The source code of this library is open source and is distributed together with QuickFit in the directory $$sourcedir$$/tinytiff. There are two C/C++ files with one header each.

  • tinytiffreader.cpp/tinytiffreader.h is a reader library for simple TIFF files
  • tinytiffwriter.cpp/tinytiffwriter.h is a fast writer library for simple TIFF files
Two test projects are provided along with the files, mentioned above (see $$sourcedir$$/tinytiff/test/).
