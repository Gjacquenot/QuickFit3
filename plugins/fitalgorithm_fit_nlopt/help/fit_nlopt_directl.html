Description

This plugin implements the locally biased DIRECT (DIviding RECTangles) algorithm for global optimization algorithm from the NLOpt library. $$ref:Gablonsky2001:J. M. Gablonsky and C. T. Kelley, "A locally-biased form of the DIRECT algorithm," J. Global Optimization, vol. 21 (1), p. 27-37 (2001).$$
$$note:The plugin uses the non-scaled variants of the algorithms.$$

$$see:See the NLopt documentation for details on this algorithm$$ $$insert:general.inc$$
