Description

This plugin implements the DIRECT (DIviding RECTangles) algorithm for global optimization algorithm from the NLOpt library. $$ref:Jones1993:D. R. Jones, C. D. Perttunen, and B. E. Stuckmann, "Lipschitzian optimization without the lipschitz constant," J. Optimization Theory and Applications, vol. 79, p. 157 (1993)$$
$$note:The plugin uses the non-scaled variants of the algorithms.$$

$$see:See the NLopt documentation for details on this algorithm$$ $$insert:general.inc$$
