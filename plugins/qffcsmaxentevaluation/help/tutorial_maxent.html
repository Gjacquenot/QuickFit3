This tutorial explains how to use the Maximum Entropy Plugin for QuickFit. This is done by providing a step by step instruction for FCS example data. It might be helpful to read the help and tutorial of the FCS fit plugin as well to get the basic idea of how data evaluation in QuickFit works.

Getting started

Once the FCS data has been loaded into the workspace we need to add an MaxEnt Evaluation object to the current project. To do so, click on the "Insert a new evaluation" button and choose "FCS: MaxEnt", or alternatively click on Data Items/Insert Evaluations/FCS: MaxEnt

Next double click on the new item in the "Evaluations Items" list in the project tree, to open the plugin environment and there switch to the "Evaluation" tab to see the MaxEnt user interface.

You should see the user interface as explained above. In the file list on the right hand side we see our opened data record which we wish to process.

MaxEnt Fitting

In this tutorial we work with the data file:

  • DNA_2nM_Al488_10ul.ASC
which can be found in the driectory output/assets/plugins/fcs_maxent/tutorial.

By following the steps in the list below you should be able to carry out a MaxEnt fit.
  1. Selecting the desired data. First of all make sure that the correct data set is chosen. Check that the data you wish to evaluate appears highlighted in the file list on the right. Then choose which run you wish to work with by using the corresponding sliders for the "run" -field.
  2. Set model parameters. To ensure a good performance it is recommended, to use the fit results from a preliminary FCS fit as input parameters for the MaxEnt fit algorithm.
    • Axial ratio: For the calibration of the axial ratio see the FCS fit help document. If not specified otherwise you might use the default value set by the algorithm.
    For the given example data we use the the model "3D diffusion with triplet." In this case a previous FCS fit provided the following values for the triplet decay time and the triplet fraction
    • τ = 9.39532 ± 0.75
    • θ = 0.2969 ± 0.0119
    We adjust the default values for these two model parameters to the newly found fit results for our given data set.
  3. Select data range. By using the lower & upper cutoff sliders below the plotting boxes we can adjust which data points we wish to neglect. In this example we will cut off the first three data points.
  4. If all values are set. We can click on the fit current button and will see both the fitted autocorrelation curve in the main plot window on the left, and the computed Maximum Entropy distribution in the plot box to the right.

A cursor function in the top toolbar above the plot boxes can be used for a rough analysis. For a further analysis of the plotted curves one should use the "Save data of the plot as file (CSV,...)" function which is highlighted with a red circle on the left.

Overlay Plots

You can also use the overlay tool to plot several MaxEnt-distributions in a single plot and compare them.
