This plugin implements a DLS diffusion model with normally distributed correlation times: $$bmath: G^{(2)}(\tau)=G_\infty+\epsilon+\left(A\cdot\frac{\int_{\tau_{min}}^{\tau_{max}}p(\tau_D)\cdot\exp\left[-\tau/\tau_D\right]\;\mathrm{d}\tau_D}{\int_{\tau_{min}}^{\tau_{max}}p(\tau_D)\;\mathrm{d}\tau_D}\right)^2$$ with a Gaussian distribution $$bmath:p(\tau_D)=\frac{1}{\sqrt{2\pi\sigma^2(\tau_D)}}\cdot\exp\left[-\frac{1}{2}\cdot\frac{(\tau_D-\tau_{c})^2}{\sigma^2(\tau_D)}\right].$$ where the parameters are:

  • $$math:G_\infty$$: offset value (usually fixed to 1)
  • $$math:\epsilon$$: offset correction
  • $$math:A$$: amplitude correction factor
  • $$math:\tau_{c}$$: center correlation time of the distribution
  • $$math:\sigma^2(\tau_D)$$: width of the distribution

$$DLS_SD_INTEGRATION$$

This plugin also calculates the apparent diffusion coefficients from a given wavelength λ, refractive index n, scattering angle Θ and the measured correlation time $$math:\tau_i$$ as $$bmath:D_i=\frac{1}{q^2\cdot\tau_i}, q=\frac{4\pi n}{\lambda}\cdot\sin(\Theta/2)$$

These formulas are used to calculate parameter errors: $$bmath:\Delta q=\sqrt{\left(\Delta n\cdot 8\pi\cdot q/\lambda\cdot\sin(\Theta/2)\right)^2+\left(\Delta\lambda\cdot 8\pi\cdot n\cdot q/\lambda^2\cdot\sin(\Theta/2)\right)^2+\left(\Delta\Theta\cdot 4\pi\cdot q/\lambda\cdot\cos(\Theta/2)\right)^2}$$ $$bmath:\Delta D_i=\sqrt{\left(\Delta q\cdot \frac{2}{q^3\cdot\tau_i}\right)^2+\left(\Delta\tau_i\cdot\frac{1}{q^2\cdot\tau_i^2}\right)^2}$$
