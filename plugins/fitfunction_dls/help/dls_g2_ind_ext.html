This plugin implements a DLS diffusion model with up to four components, written in terms of the diffusion coefficient, not the correlation decay time: $$bmath: G^{(2)}(\tau)=G_\infty+\epsilon+\left(\sum_{i=1}^8a_i\cdot e^{-q^2\cdot D\cdot\tau}\right)^2, q=\frac{4\pi n}{\lambda}\cdot\sin(\Theta/2)$$ where the parameters are:

  • $$math:G_\infty$$: offset value (usually fixed to 1)
  • $$math:\epsilon$$: offset correction
  • $$math:a_i$$: fraction of specied i
  • $$math:D_i$$: apparent diffusion coefficient time of species i
  • $$math:q=\frac{4\pi n}{\lambda}\cdot\sin(\Theta/2)$$: scattering vector
  • $$math:\lambda$$: wavelength of scattered light
  • $$math:n$$: solution's refractive index
  • $$math:\Theta$$: scattering angle

This plugin also calculates the correlation decay time from a measured diffusion coefficient $$math:D_i$$ as $$bmath:\tau_i=\frac{1}{q^2\cdot D_i}$$

These formulas are used to calculate parameter errors: $$bmath:\Delta q=\sqrt{\left(\Delta n\cdot 8\pi\cdot q/\lambda\cdot\sin(\Theta/2)\right)^2+\left(\Delta\lambda\cdot 8\pi\cdot n\cdot q/\lambda^2\cdot\sin(\Theta/2)\right)^2+\left(\Delta\Theta\cdot 4\pi\cdot q/\lambda\cdot\cos(\Theta/2)\right)^2}$$ $$bmath:\Delta \tau_i=\sqrt{\left(\Delta q\cdot \frac{2}{q^3\cdot D_i}\right)^2+\left(\Delta D_i\cdot\frac{1}{q^2\cdot D_i^2}\right)^2}$$
