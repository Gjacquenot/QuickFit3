$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin displays the contents of a settings/ini/configuration file in the following format:
[Group1]
group11/keya=abc
group11/keyb=1234.56
group12/keya=xyz
keya=12
keyb=xyz

[Group2]
group21/keya=abc
group21/keyb=1234.56
keya=12
keyb=xyz
