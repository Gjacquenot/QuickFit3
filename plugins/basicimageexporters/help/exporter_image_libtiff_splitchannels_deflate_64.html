This exporter saves an image series into a TIFF file, using loss-less ZIP/Deflate compression. It also splits the channels into separate files. It will create one file BASENAME_chXX.tif for each color channel, where XX is the channel number (0,1,...) and BASENAME is the filename (without extension), which was given by the user.
