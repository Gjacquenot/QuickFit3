This exporter saves an image series I(x, y, z, c) into a RAW file, which contains only the data as a 3-dimensional array in row-major order. Each entry is a 64-bit double-precision floating-point number, written in little-endian (PC standard) form. The channels are written separately for each frame: 

 // frame 0, channel 0
 I(0,0,0,0), I(1,0,0,0), I(2,0,0,0), ..., I(width-1,0,0,0)
 I(0,1,0,0), I(1,1,0,0), I(2,1,0,0), ..., I(width-1,1,0,0)
 ......
 I(0,height-1,0,0), I(1,height-1,0,0), I(2,height-1,0,0), ..., I(width-1,height-1,0,0)

 // frame 0, channel 1
 I(0,0,0,1), I(1,0,0,1), I(2,0,0,1), ..., I(width-1,0,0,1)
 I(0,1,0,1), I(1,1,0,1), I(2,1,0,1), ..., I(width-1,1,0,1)
 ......
 I(0,height-1,0,1), I(1,height-1,0,1), I(2,height-1,0,1), ..., I(width-1,height-1,0,1)
 ......
 I(0,height-1,0,channels-1), I(1,height-1,0,channels-1), I(2,height-1,0,channels-1), ..., I(width-1,height-1,0,channels-1)
 
 // frame 1, channel 0
 I(0,0,1,0), I(1,0,1,0), I(2,0,1,0), ..., I(width-1,0,1,0)
 I(0,1,1,0), I(1,1,1,0), I(2,1,1,0), ..., I(width-1,1,1,0)
 ......
 I(0,height-1,1,0), I(1,height-1,1,0), I(2,height-1,1,0), ..., I(width-1,height-1,1,0)
 
 // frame 1, channel 1
 ......
 
 // last frame, last channel ...
 I(0,height-1,depth-1,channels-1), I(1,height-1,depth-1,channels-1), I(2,height-1,depth-1,channels-1), ..., I(width-1,height-1,depth-1,channels-1)
