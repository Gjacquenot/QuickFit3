This exporter saves an image series I(x, y, z, c) as a Matlab matrix with dimensions (X,Y,Z,C), or if a single multi-color frame is written as (X,Y,C), or if a single-channel image series is written as (X,Y,Z) into a Matlab .mat data file. You can open the file in Matlab (or also GNU Octave) with a script, such as this (for a single-channel image stack): 

load('filename');
imagesc(image(:,:,50)); % QuickFit saves the image stack as a variable 
                        % named "image", here we display the 50-th frame!
The following script reads a multi-channel and multi-frame image stack and displays the 50th frame from the first and second channe: 
load('filename');
subplot(1,2,1)
imagesc(image(:,:,50,1))
colormap gray
colorbar

subplot(1,2,2)
imagesc(image(:,:,50,2))
colormap gray
colorbar
$$note:This exporter uses the TinyMAT library from https://github.com/jkriege2/TinyMAT$$
