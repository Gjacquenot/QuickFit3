This exporter saves an image series into a TIFF file, using lossy JPEG compression. It also splits the channels into separate files. It will create one file BASENAME_chXX.tif for each color channel, where XX is the channel number (0,1,...) and BASENAME is the filename (without extension), which was given by the user. $$note:This exporter will convert 64-bit double-images to a 32-bit float, since 64-bit samples are not supported by all image processing tools (e.g. IMageJ does not support them!)$$
