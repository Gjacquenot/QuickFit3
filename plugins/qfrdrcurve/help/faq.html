$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ $$faq_start$$ How convert a curve RDR to a table RDR? $$faq_answer$$
  1. open the curve's RDR editor by double-clicking in the project tree.
  2. switch to the tab "XY-Curves editor"
  3. right-click the plot and choose "copy plot to table RDR" from the context menu. Fill in the form and click on "OK"
$$faq_end$$ $$faq_start$$ How see a data-table for a curve-RDR? $$faq_answer$$
  1. open the curve's RDR editor by double-clicking in the project tree.
  2. switch to the tab "XY-Curves editor"
  3. right-click the plot and choose "Show Plot Data" from the context menu. A window with the data will open.
$$faq_end$$
