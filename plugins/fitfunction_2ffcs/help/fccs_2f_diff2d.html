This plugin implements a 3D diffusion FCS fit function for a dual-focus setting, as described in:

The model function is: $$bmath:g(\tau)= \frac{1}{N}\cdot\left(\frac{I_1-B_1}{I_1}\right)\cdot\left(\frac{I_2-B_2}{I_2}\right)\cdot\left[(1-\rho_2-\rho_3)\cdot g_1(\tau)+\rho_2\cdot g_2(\tau)+\rho_3\cdot g_3(\tau)\right]+G_\infty \text{with} g_i(\tau)=\left(1+\frac{4D_i\tau}{w_{xy}^2}\right)^{-1}\cdot\exp\left(-\frac{d_{xy}^2}{w_{xy}^2+4D\tau}\right)$$ Here the parameters are defined as follows:
  • $$math:N$$: apparent particle number in the sample
  • $$math:D_{i}$$: diffusion coefficient of species i
  • $$math:w_{xy}$$: lateral $$math:1/e^2$$-radius (i.e. half-width) of the PSFs
  • $$math:d_{xy}$$: lateral displacement of the two foci
  • $$math:I_1, I_2$$: count rates, as measured in two foci (including background contribution!)
  • $$math:B_1, B_2$$: background count rates in the two foci
  • $$G_\infty$$: offset of correlation function (usually fixed to 0 or 1)
  • $$math:\rho_i$$ fraction of particles of species i. Note that the $$math:\sum_i\rho_i=1$$

This model also calculates the diffusion time (retention time in the focus), if the width $$math:w_{y}$$ of the laser focus (xy plane) is known, as: $$bmath:\tau_D=\frac{w_{xy}^2}{4D}$$ where $$math:D$$ is the diffusion coefficient. Also the effective focal area $$math:A_{eff}$$ is calculated:

$$bmath:A_{eff}=\pi\cdot w_{xy}^2$$ Given this focal volume, the plugin may also calculate the particle concentration in the sample:

$$bmath:C=\frac{N}{V_{eff}}$$ Given the average count rate $$math:\overline{I}=(I_1+I_2)/2$$ and the average background count rate $$math:\overline{B}=(B_1+B_2)/2$$, we can calculate the counts per molecule: $$bmath:\text{CPM}=\frac{\overline{I}-\overline{B}}{N}$$ The plugin also calculates some parameter errors:
$$bmath:\Delta \left(\frac{1}{N}\right)=\left|\frac{\Delta N}{N^2}\right|$$ $$bmath:\Delta\tau_D=\sqrt{\left(\Delta w_{xy}\cdot\frac{w_{xy}}{2D}\right)^2+\left(\Delta D\cdot\frac{w_{xy}^2}{4D^2}\right)^2}$$ $$bmath:\Delta A_{eff}=\sqrt{\left(\Delta\gamma\cdot2\pi\cdot w_{xy}right)^2}$$ $$bmath:\Delta\text{cpm}=\sqrt{\left(\Delta I_1\cdot\frac{1}{2N}\right)^2+ \left(\Delta I_2\cdot\frac{1}{2N}\right)^2 +\left(\Delta B_1\cdot\frac{1}{2N}\right)^2 +\left(\Delta B_1\cdot\frac{1}{2N}\right)^2 + \left(\Delta N\cdot\frac{I_1+I_2-(B_1+B_2)}{2N^2}\right)^2}$$
