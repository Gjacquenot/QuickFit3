FCS/DLS Curve Simulator

Usage

This tool allows create raw data records with simulated correlation data. To create a record with synthetic data, use the menu entry "Data Items|Insert Raw Data|FCS/DLS data|FCS/DLS Data from fit function". Then a dialog will pop up, where you can define all parameters for the curves. After clicking "OK", a record with the data as in the dialog will be created and inserted into the project.

Parametrizing the synthetic correlation curve

You can set these parameters for a synthetic correlation curve:
  • model function: select the model function $$math:g(\tau, \vec{p})$$ from which you want to create synthetic data
  • parameters: Here you can parametrize the model function
  • lag time range: specify the minimum and maximum lag time $$math:\tau$$ for which you want to evaluate the model function
  • runs: This plugin may create several runs. All runs will have the same model parameters, but the may differ due to noise.
  • noise: Here you can choose a noise model and parameters for your synthetic data:
    • Koppel's formula: Koppel derived a formula foe the variance of the ACF, which is evaluated here (see Wohland, T; Rigler, R; Vogel, H (2001): "The standard deviation in fluorescence correlation spectroscopy", Biophys J 80(6), p. 2987-99). In addition to the model parameters you will have to supply the duration of the measurement and the average photon count during the measurement.
    • gaussian: adds gaussian noise with a standard deviation which is calculated as a fraction of the $$math:g(\tau)$$ at the minimum lag time. The fraction is given by the parameter "level".
    • uniform: adds uniform noise with a width which is calculated as a fraction of the $$math:g(\tau)$$ at the minimum lag time. The fraction is given by the parameter "level".
    • local gaussian: like gaussian, but the noise scales with the local value of the ACF.
    • local uniform: like uniform, but the noise scales with the local value of the ACF.
