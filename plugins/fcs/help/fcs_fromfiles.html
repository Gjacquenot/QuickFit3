Open Correlation Data from Files

This plugin allows to open Fluorescence (Cross-)Correlation Spectroscopy (FCS/FCCS) data into QuickFit. These dataformats are readable:
  • ALV-5000
  • ISS Alba Files
  • ASCII Data Files (e.g. Comma Separated Values)
