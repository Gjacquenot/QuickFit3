This function implements a 1-component anomalous diffusion model for data acquired with a camera in TIR illumination mode. It assumes that the time series of two distant pixels were cross-correlated. $$ref:Sankaran2013:Sankaran, J.; Bag, N.; Kraut, R.S.; Wohland, T. Accuracy and precision in camera-based fluorescence correlation spectroscopy measurements, Anal. Chem. 85 (8) (2013) 3948-54. $$ $$ref:Sankaran2010:Sankaran, J.; Shi, X.; Ho, L.Y.; Stelzer, E.H.; Wohland, T. ImFCS: a software for imaging FCS data analysis and visualization, Opt Express, 18 (2010) 25468-25481$$ The fit function is defined as: $$bmath:g(\tau)=G_\infty + \frac{(I-B)^2}{I^2}\cdot\frac{1}{N/A_{\text{eff}}}\cdot\sum_{i=1}^3\rho_i\cdot g_{xy}(\tau; v_x,\Gamma_i,\alpha_i,d_x)\cdot g_{xy}(\tau; v_y,\Gamma_i,\alpha_i,d_y),\ \ \ \ \ \sum_{i=1}^3\rho_i=1$$ $$bmath:\mathrm{with:}\ \ \ g_{xy}(\tau;v,\Gamma,\alpha,d)=\frac{1}{2a^2}\cdot\left\{ \left[(a-v_x\tau+d)\cdot\erf\left(\frac{a-v_x\tau+d}{\sqrt{4\Gamma\tau^\alpha+w_{xy}^2}}\right)-2(v_x\tau-d)\cdot\erf\left(\frac{v_x\tau-d}{\sqrt{4\Gamma\tau^\alpha+w_{xy}^2}}\right)+(a+v_x\tau-d)\cdot\erf\left(\frac{a+v_x\tau-d}{\sqrt{4\Gamma\tau^\alpha+w_{xy}^2}}\right)\right]+\frac{\sqrt{4\Gamma\tau^\alpha+w_{xy}^2}}{\sqrt{\pi}}\cdot\left[\mathrm{e}^{-\frac{(a-v_x\tau+d)^2}{4\Gamma\tau^\alpha+w_{xy}^2}}-2\cdot\mathrm{e}^{-\frac{(v_x\tau-d)^2}{4\Gamma\tau^\alpha+w_{xy}^2}}+\mathrm{e}^{-\frac{(a+v_x\tau-d )^2}{4\Gamma\tau^\alpha+w_{xy}^2}}\right]\right\}$$ $$bmath:A_{\text{eff}}=\frac{a^2}{\left(\mbox{erf}\left(\frac{a}{w_{xy}}\right)+\frac{w_{xy}}{\sqrt{\pi}\cdot a}\cdot\left(e^{-(a/w_{xy})^2}-1\right)\right)^2}$$ The parameters are:
