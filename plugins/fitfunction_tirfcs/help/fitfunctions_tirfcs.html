Fit Functions for TIR-FCS/FCCS

$$see:There is also a frequently asked questions (FAQ) page, which explains which TIR-FCS/FCCS fit function to choose for which case$$
  • FCS, i.e. single-pixel autocorrelation:
    • diffusion (1/e2 width)
    • diffusion, different molecular brightnesses (1/e2 width)
    • anomalous diffusion (1/e2 width)
    • two-pixel FCCS:
      • diffusion+flow (1/e2 width)
    • two-color FCCS:
      • 1-component normal diffusion:
        • TIR-FCCS: 1-comp. 2D diffusion (xy), species A+B+AB, c/D per species, ACF green (1/e² radii)
        • TIR-FCCS: 1-comp. 2D diffusion (xy), species A+B+AB, c/D per species, ACF red (1/e² radii)
        • TIR-FCCS: 1-comp. 2D diffusion (xy), species A+B+AB, c/D per species, CCF (1/e² radii)
      • 2-component normal diffusion:
        • TIR-FCCS: 2-comp. 2D diffusion (xy), species A+B+AB, c per species, D1/D2 per channel, ACF green (1/e² radii)
        • TIR-FCCS: 2-comp. 2D diffusion (xy), species A+B+AB, c per species, D1/D2 per channel, ACF red (1/e² radii)
        • TIR-FCCS: 2-comp. 2D diffusion (xy), species A+B+AB, c per species, D1/D2 per channel, CCF (1/e² radii)
      • 1-component anomalous diffusion:
        • TIR-FCCS: 1-comp. 2D anomalous diffusion (xy), species A+B+AB, c/D per species, ACF green (1/e² radii)
        • TIR-FCCS: 1-comp. 2D anomalous diffusion (xy), species A+B+AB, c/D per species, ACF red (1/e² radii)
        • TIR-FCCS: 1-comp. 2D anomalous diffusion (xy), species A+B+AB, c/D per species, CCF (1/e² radii)
