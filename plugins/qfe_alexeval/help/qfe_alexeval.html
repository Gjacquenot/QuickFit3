Description

This plugin can be sued to perform an spFRET for single-photon data acquired with an alternating laser-excitation (ALEX) setup. At the moment this tool can only read v5.x/6.x TTTR-data from a PicoQuant TimeHarp 200 card (*.t3r-files). You can start the plugin by clicking on the menu entry Extensions | Start ALEX Evaluation (for spFRET) in QuickFit's main widnow.
