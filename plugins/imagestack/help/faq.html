$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ $$faq_start$$ How can I export an image stack? $$faq_answer$$ You can use the general export tool of the image stack plugin, either by right-clicking an image stack RDR in the project tree (in the main window) and choosing Export Data From RDR, or by clicking on the menu entry Record | Export Data in the image stack RDR editor window. In both cases, you will be asked to select a new filename and an export dataformat. Several export formats are available (e.g. several TIFF variants). $$faq_end$$
