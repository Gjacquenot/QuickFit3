$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin implements these fit functions for Dual-color fluorescence cross corelation spectroscopy (FCCS):
  • FCCS 3D Diffusion
  • 2-color crosscorrelation framework, 1-component normal diffusion:
    • ACF, green
    • ACF, red
    • CCF
  • 2-color crosscorrelation framework, 1-component normal diffusion, c per species, D per channel:
    • ACF, green
    • ACF, red
    • CCF
  • 2-color crosscorrelation framework, 2-component normal diffusion:
    • ACF, green
    • ACF, red
    • CCF
  • 2-color crosscorrelation framework, 2-component normal diffusion, c per species, D1/D2 per channel:
    • ACF, green
    • ACF, red
    • CCF
  • 2-color crosscorrelation framework, 1-component anomalous diffusion:
    • ACF, green
    • ACF, red
    • CCF
