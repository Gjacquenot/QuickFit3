SPIM-FCCS: Diffusion, 2-color, CCF (1/e² radii)

$$see:ACF, green model, ACF, red model, CCF, model$$

This function implements a 1-component diffusion model $$math:G_{gr}(\tau)$$ for 2-color crosscorrelation data acquired with a camera in SPIM illumination mode. The theory and functional form are described below. In addition these quantities are calculated by the model: $$bmath:\frac{c_{AB}}{c_{all}}=\frac{c_{AB}}{c_A+c_B+c_{AB}},\ \ \ \ \ \Delta\left(\frac{c_{AB}}{c_{all}}\right)=\sqrt{\left(\Delta c_A\cdot\frac{c_{AB}}{(c_A+c_B+c_{AB})^2}\right)^2+\left(\Delta c_B\cdot\frac{c_{AB}}{(c_A+c_B+c_{AB})^2}\right)^2+\left(\Delta c_{AB}\cdot\frac{c_A+c_B}{(c_A+c_B+c_{AB})^2}\right)^2}$$ $$bmath:\frac{c_{AB}}{\min(c_A, c_B)}$$ $$bmath:K_d=\frac{c_A\cdot c_B}{c_{AB}},\ \ \ \ \ \Delta K_d=\sqrt{\left(\Delta c_A\cdot\frac{c_B}{c_{AB}}\right)^2+\left(\Delta c_B\cdot\frac{c_A}{c_{AB}}\right)^2+\left(\Delta c_{AB}\cdot\frac{c_A\cdot c_B}{c_{AB}^2}\right)^2}$$
