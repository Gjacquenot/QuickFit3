$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin adds support for export to LaTeX to the graph plotter used by many plugins in QuickFit 3:
  • Enhanced Metafile Format (EMF): this is currently supported on windows only
  • LaTeX [...]:
    • Tikz: outputs Tikz code
    • Pgf: outputs Pgf code
    • b&w: colors are converted to black and white, if this option is not present, the output is in color
    • include file: This is the default output format, in which the produced .tex file can be included in another file with \include{...} to place the figure, where you want it.
    • full document: If this option is present, the .tex file will contain a full LaTeX header and can be compiled as a standalone file with latex directly.
    • no TeX parser: If this option is present, the LaTeX strings in a plot will not be rendered, but output directly into the document, so the text is completely typeset by LaTeX. $$note:In this mode, the text may not completely resemble the presentation in QuickFit!$$
