Introduction

$$insert:dist_basics.inc$$

Some older version of these models (marked as deprecated!) implement the integrals as a sum over semi-logarithmically spaced diffusion times $$math:\tau_D\in\{\tau_{min}...\tau_{max}\}$$.

Models like these are defined/used e.g. in $$REF_PAL_FCSDIST$$.

List of available models

  • models, using numerical integration:
    • $$fitfunction:name:fcs_dist_int_norm$$
    • $$fitfunction:name:fcs_dist_int_lognorm$$
    • $$fitfunction:name:fcs_dist_int_norm_d$$
  • deprecated models, replaced by models using numerical integration:
    • $$fitfunction:name:fcs_dist_norm_d$$
    • $$fitfunction:name:fcs_dist_norm$$
    • $$fitfunction:name:fcs_dist_lognorm$$
