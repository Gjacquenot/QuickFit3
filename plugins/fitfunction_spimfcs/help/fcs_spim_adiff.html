SPIM-FCCS fit model: anomalous diffusion, 1/√e-radii

This function implements a 1..3-component anomalous diffusion model for data acquired with a camera in SPIM illumination mode. The fit function is defined as: $$bmath:g(\tau)=G_\infty+\frac{1}{N}\cdot\frac{1}{4a^2\sqrt{\pi}}\cdot\sum\limits_{i=1}^3\rho_i\cdot\left[2a\cdot\mbox{erf}\left(\frac{a}{2\cdot\sqrt{\Gamma_i\tau^{\alpha_i}+\sigma_{xy}^2}}\right)+\frac{4\cdot\sqrt{\Gamma_i\tau^{\alpha_i}+\sigma_{xy}^2}}{\sqrt{\pi}}\cdot\left(\exp\left(-\frac{a^2}{4\cdot(\Gamma_i\tau^{\alpha_i}+\sigma_{xy}^2)}\right)-1\right)\right]^2\cdot\left[1+\frac{\Gamma_i\tau^{\alpha_i}}{\sigma_z^2}\right]^{-1/2}$$ The parameters are:

var.Parameter
G∞constant offset
Nparticle number in the observation volume
Γii-th anomalous diffusion coefficient: $$math:\langle r^2(\tau)\rangle=6\cdot\Gamma_i\cdot\tau^{\alpha_i}=6\cdot\frac{K_i\cdot\tau^{\alpha_i}}{\Gamma(1+\alpha)} $$
αii-th anomality parameter
ρifraction of i-th diffusing component
awidth of a (square) pixel in the image plane (i.e. pixel size of camera divided by the magnification)
σxylateral radius of the point-spread function (PSF), 1/√e-radius
σzlongitudinal radius of the point-spread function (PSF), 1/√e-radius
