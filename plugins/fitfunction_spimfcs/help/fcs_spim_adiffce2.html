SPIM-FCS fit model anomalous diffusion, c as fit parameter (1/e2-radii, new Veff)

This function implements a 1..3-component anomalous diffusion model for data acquired with a camera in SPIM illumination mode. The fit function is defined as: $$bmath:g(\tau)=G_\infty+\frac{1}{C}}\cdot\frac{1}{\sqrt{\pi}\cdot w_za^2}\cdot\sum\limits_{i=1}^3\rho_i\cdot\left[\mbox{erf}\left(\frac{a}{\sqrt{4\Gamma_i\tau^{\alpha_i}+w_{xy}^2}}\right)+\sqrt{4\Gamma_i\tau^{\alpha_i}+w_{xy}^2}\cdot\left[\exp\left(-\frac{a^2}{4\Gamma_i\tau^{\alpha_i}+w_{xy}^2}\right)-1\right]\right]^2\cdot\left[1+\frac{4\Gamma_i\tau^{\alpha_i}}{w_z^2}\right]^{-1/2}$$ $$bmath:V_{\text{eff}}=\frac{\sqrt{\pi}\cdot a^2w_z}{\left(\mbox{erf}\left(\frac{a}{w_{xy}}\right)+\frac{w_{xy}}{\sqrt{\pi}\cdot a}\cdot\left(e^{-(a/w_{xy})^2}-1\right)\right)^2}$$ The parameters are:

var.Parameter
G∞constant offset
Cparticle concentration. The particle number in the observation volume is $$math:N=C\cdot V_{eff}$$
Γii-th anomalous diffusion coefficient: $$math:\langle r^2(\tau)\rangle=6\cdot\Gamma\cdot\tau^\alpha=6\cdot\frac{K}{\Gamma (1+\alpha )} \cdot\tau^{\alpha} $$
αii-th anomality parameter
ρifraction of i-th diffusing component
awidth of a (square) pixel in the image plane (i.e. pixel size of camera divided by the magnification)
wxylateral radius of the point-spread function (PSF), 1/e2-radius
wzlongitudinal radius of the point-spread function (PSF), 1/e2-radius
$$math:I$$average measured intensity in focus during measurement
$$math:B$$average background intenisty in focus during measurement
For the fractions we have in addition: $$bmath:\rho_1=1-\rho_2-\rho_3$$ $$bmath:\Delta\rho_1=\sqrt{\Delta\rho_2^2+\Delta\rho_3^2}$$ $$bmath:N=C\cdot V_{eff}$$ $$bmath:\Delta N=C\cdot V_{eff}$$
