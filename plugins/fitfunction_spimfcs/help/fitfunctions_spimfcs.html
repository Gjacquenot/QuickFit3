    • anomalous diffusion, c as fit parameter ($$math:1/e2$$ width, new Veff)
    • normal diffusion (1/e2 width)
    • normal diffusion (1/e2 width, new Veff)
    • normal diffusion, different molecular brightnesses per species (1/e2 width, new Veff)
    • normal diffusion, c&D as fit parameters (1/e2 width)
    • 3D normal Diffusion+Flow (1/e2 width, new Veff)
    • Deprecated: normal diffusion ($$math:1/\sqrt{e}$$ width)

    • Deprecated: anomalous diffusion ($$math:1/\sqrt{e}$$ width)
