Fit Functions for SPIM-FCS

  • FCS, i.e. single-pixel autocorrelation:
    • anomalous diffusion ($$math:1/e^2$$ width, new Veff)
    • anomalous diffusion, c as fit parameter ($$math:1/e^2$$ width, new Veff)
    • normal diffusion (1/e2 width)
    • normal diffusion (1/e2 width, new Veff)
    • normal diffusion, different molecular brightnesses per species (1/e2 width, new Veff)
    • normal diffusion, c&D as fit parameters (1/e2 width)
    • 3D normal Diffusion+Flow (1/e2 width, new Veff)
    • Deprecated: normal diffusion ($$math:1/\sqrt{e}$$ width)
    • Deprecated: normal diffusion, c&D as fit parameters ($$math:1/\sqrt{e}$$ width)
    • Deprecated: anomalous diffusion ($$math:1/\sqrt{e}$$ width)
  • two-pixel cross-correlation:
    • normal diffusion+flow ($$math:1/e^2$$ width)
    • normal diffusion ($$math:1/e^2$$ width)
    • anomalous diffusion ($$math:1/e^2$$ width)
    • diffusion+flow+global decay ($$math:1/e^2$$ width)
    • Deprecated: normal diffusion ($$math:1/\sqrt{e}$$ width)
    • Deprecated: normal diffusion+flow ($$math:1/\sqrt{e}$$ width)
    • Deprecated: 2-color crosscorrelation normal diffusion ($$math:1/\sqrt{e}$$ width)
  • 2-color cross-correlation:
    • 2-color crosscorrelation framework, normal diffusion (1/e2 radii):
      • ACF, green
      • ACF, red
      • CCF
    • 2-color crosscorrelation framework, normal diffusion, factor D (1/e2 radii): In this model $$math:D_{b}=F_b\cdot D_a$$ and $$math:D_{ab}=F_{ab}\cdot D_a$$
      • ACF, green
      • ACF, red
      • CCF
    • 2-color crosscorrelation framework, 2-component normal diffusion (1/e2 radii):
      • ACF, green
      • ACF, red
      • CCF
    • 2-color crosscorrelation framework, anomalous diffusion (1/e2 radii):
      • ACF, green
      • ACF, red
      • CCF
