SPIM-FCCS fit model: Diffusion 2-color crosscorrelation (1/$$math:\sqrt(e)$$ radii)

This function implements a 1-component diffusion model for 2-color crosscorrelation data acquired with a camera in SPIM illumination mode. The fit function is defined as: $$bmath:g(\tau)=G_\infty+\frac{1}{N}\cdot\frac{1}{4a^2\sqrt{\pi}}\cdot\sum\limits_{i=1}^3\rho_i\cdot\left[2a\cdot\mbox{erf}\left(\frac{a}{2\cdot\sqrt{\Gamma_i\tau^{\alpha_i}+\sigma_{xy}^2}}\right)+\frac{4\cdot\sqrt{\Gamma_i\tau^{\alpha_i}+\sigma_{xy}^2}}{\sqrt{\pi}}\cdot\left(\exp\left(-\frac{a^2}{4\cdot(\Gamma_i\tau^{\alpha_i}+\sigma_{xy}^2)}\right)-1\right)\right]^2\cdot\left[1+\frac{\Gamma_i\tau^{\alpha_i}}{\sigma_z^2}\right]^{-1/2}$$ The parameters are:

G∞constant offset
$$math:N_{ab}$$particle number of the crosscorrelating species the observation volume
$$math:D_{ab}$$diffusion coefficient of the crosscorrelating species
$$math:a$$width of a (square) pixel in the image plane (i.e. pixel size of camera divided by the magnification)
$$math:\sigma_{1,xy}$$lateral radius of the point-spread function (PSF) of the first focus (e.g. green), $$math:1/\sqrt(e)$$-radius
$$math:\sigma_{1,z}$$longitudinal radius of the point-spread function (PSF) of the first focus (e.g. green), $$math:1/\sqrt(e)$$-radius
$$math:\sigma_{2,xy}$$lateral radius of the point-spread function (PSF) of the second focus (e.g. green), $$math:1/\sqrt(e)$$-radius
$$math:\sigma_{2,z}$$longitudinal radius of the point-spread function (PSF) of the first second (e.g. green), $$math:1/\sqrt(e)$$-radius
