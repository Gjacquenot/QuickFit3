$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ This plugin allows to export data from QuickFit tables into several additional data formats. It uses several open source libraries, listed here. Currently these file formats are supported:
  • Excel 97 Worksheet (*.tex): There is also an option to save the table in a flipped, or pivoted version (i.e. rotated by 90°).
