Introduction

The supported LaTeX subset is listes below. Please note that some commands are used differently than in actual LaTeX. For example \bf is just a renamed form of \textbf and used as \bf{...} and NOT as {\bf...}.

List of available commands

Here is a (hopefully) complete list of the usable commands. More commands might work, but are not guaranteed.

  • $ : enter/leave math mode
  • \bf{...} \textbf{...} \mathbf{...}: draw the contained text in bold face
  • \it{...} \textit{...} \mathit{...} : draw the contained text in italic face
  • \rm{...} \textrm{...} \mathrm{...} \mbox{...} : draw the contained text in normal upright roman font face
  • \sf{...} \textsf{...} \mathsf{...} : draw the contained text in normal upright sans-serif font face
  • \script{...} \textscript{...} \mathscript{...} : draw the contained text in a script font face
  • \textcolor{COLOR}{...} \mathcolor{COLOR}{...} \color{COLOR}{...}: draw the contained text in the given COLOR (all SVG color names are allowed!)
  • \sc{...} : draw the text in small caps
  • \ul{...} \underline{...} \underlined{...} : draw the text with underlining
  • \uul{...} \uuline{...} : draw the text with double underlining
  • \ol{...} \overline{...} \overlined{...} : draw the text with overlining
  • \ool{...} \ooline{...} : draw the text with double overlining
  • \tt{...} \texttt{...} \mathtt{...} : draw text in typewriter font
  • \fbox{...} \boxed{...} : draw a box around the text
  • \colorbox{COLOR}{...} : draw a box with the given linecolor around the text
  • \alpha ... : display the according greek letter
  • ^{...} : display the contents of braces in superscript
  • _{...} : display the contents of braces in subscript
  • \{ / \} : display opening/closing brace
  • \vec{...} : text with vector arrow over it: $$math:\vec{x}$$
  • \hat{...} : text with hat over it: $$math:\hat{x}$$
  • \arrow{...} : text with arrow over it: $$math:\arrow{x}$$
  • \dot{...} : text with derivative dot over it: $$math:\dot{x}$$
  • \ddot{...} : text with double derivative dots over it: $$math:\ddot{x}$$

  • \frac{x}{y} : shows a fraction: $$math:\frac{x}{y}$$
  • \binom{x}{y} : shows a binomial: $$math:\binom{x}{y}$$
  • \stackrel{x}{y} : shows a stack: $$math:\stackrel{x}{y}$$
  • \underbrace{xyz}{abc} : shows a underbraced text: $$math:\underbrace{xyz}{abc}$$
  • \overbrace{xyz}{abc} : shows a overbraced text: $$math:\overbrace{xyz}{abc}$$
  • \begin{array}c_{11}&c_{12}\\c_{21}&c_{22}\end{array} : shows a table: $$math:\begin{array}c_{11}&c_{12}\\c_{21}&c_{22}\end{array}$$
  • \begin{cases}c_{11}&c_{12}\\c_{21}&c_{22}\end{cases} : shows cases: $$math:\begin{cases}c_{11}&c_{12}\\c_{21}&c_{22}\end{cases}$$
  • \begin{pmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{pmatrix} : shows a braced matrix: $$math:\begin{pmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{pmatrix}$$
  • \begin{bmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{bmatrix} : shows a braced matrix: $$math:\begin{bmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{bmatrix}$$
  • \begin{Bmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{Bmatrix} : shows a braced matrix: $$math:\begin{Bmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{Bmatrix}$$
  • \begin{vmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{vmatrix} : shows a braced matrix: $$math:\begin{vmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{vmatrix}$$
  • \begin{Vmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{Vmatrix} : shows a braced matrix: $$math:\begin{Vmatrix}c_{11}&c_{12}\\c_{21}&c_{22}\end{Vmatrix}$$

Braces in math mode are adjusted in size, so they are a small bit larger than the contents. To enable this you have to write braces with \left and \right. These types of braces are defined (slight differences to LaTeX standard):

  • \left( \right) : default meaning ()
  • \left[ \right] : default meaning []
  • \left\{ \right\} : default meaning {}
  • \left< \right> : "averaging" braces <>
  • \left \right : floor braces |_ _|
  • \left~ \right~ : ceil braces |~ ~|
  • \left| \right| : absolute value braces | |
  • \left\| \right\| : norme braces || ||
