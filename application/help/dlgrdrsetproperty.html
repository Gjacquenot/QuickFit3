Set RDR Property Dialog

After clicking on $$bqtt:Tools | project tools | Set property in multiple RDRs$$ a dialog opens that allows to set/add an RDR property to several (selectable) raw data records. Instead of an RDR property, it is also possible to set the name, folder, group of the selected RDRs (select in "property type")

  1. Simply select the RDRs to which the property should be added/where it should be set in the list on the rhs.
  2. Set the property name, value (true/false for boolean!) and data-type
  3. Set the other applicable options and click OK.
