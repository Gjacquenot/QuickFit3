Set RDR Property Dialog

After clicking on $$bqtt:Tools | project tools | Set property in multiple RDRs$$ a dialog opens that allows to set/add an RDR property to several (selectable) raw data records.

  1. Simply select the RDRs to which the property should be added/where t should be set in the list on the rhs.
  2. Set the property name, value and type (true/false for boolean!)
  3. Set the applicable options and click OK.
