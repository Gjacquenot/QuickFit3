Introduction

If you need a C/C++ compiler on Microsoft Windows, which can be used to create plugins for QuickFit $$version$$ (e.g. for user-defined fit functions), you can use the free MinGW/MinGW-W64 version of the GNU Compiler Collection (GCC). A project, which distributes pre-compiled packages of this compiler, together with the needed libraries is MinGW-builds. This is also the compiler, that is used to build the windows binary releases of QuickFit $$version$$.

$$note:QuickFit is available in two flavors: a 32-bit and a 64-bit version and if you write plugins, the compiler has to produce binaries, which match the bit-depth of your QuickFit copy. The bit-depth is displayed at the bottom of the QuickFit-welcome screen/splash window, which you can view in the about-dialog (main-window, menu entry: Help | About). It is also listed under "QuickFit bit-depth" in the "Info"-tab of that about-dialog$$ $$note:If you need to know the exact name/version of the compiler, used to build you QuickFit version, you can look into the about-dialog (main-window, menu entry: Help | About). There the compiler is listed in the "Info" tab, if you scroll down to the entry "compiler used for this version:".$$

How-to install

  1. Find out the bit-depth of your QuickFit version (see above).
  2. Go to the webpage http://sourceforge.net/projects/mingwbuilds/ and download the installer there.
