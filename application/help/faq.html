How to define your own fit-functions?
You can define fit functions via the internal math expression parser (simple), or via simple fit function plugins (faster). A detailed description can be found under user-defined fit functions.
How to save a plot as an image/PDF/LaTeX?
You can save most plots in QuickFit by right-clicking on the plot and then choosing Save Plot. A detailed description can be found under $$qf_ui_jkqtplotter_helpfiletitle$$.
How to print a plot?
You can print most plots in QuickFit by right-clicking on the plot and then choosing Print Plot. A detailed description can be found under $$qf_ui_jkqtplotter_helpfiletitle$$.
How to save a plot as an editable table-RDR in a project?
Most plots in QuickFit can be stored as a table raw data record in the current project. QuickFit then creates a new (or appends to an existing) table RDR in the current project, adds all data columns, that are necessary for the plot to that table and creates a plot in it, which resembles the plot you want to save. This way, the plot can be altered and formatted (e.g. for a publication). ALso the data can be procesed further (fits, statistics etc.).
To copy a plot in this way,
  1. right-click the plot
  2. choose copy plot data to table RDR
  3. set all properties in the following dialog as required (e.g. give a new name for the table-RDR, or select an existing one and give a new name for the plot in the table (or select an existing one).
  4. now you can edit the plot (and its data), by double-clicking the new table RDR in the project tree.
A detailed description can be found under $$qf_ui_jkqtplotter_helpfiletitle$$.

How do I rename RDR groups for several RDRs?
You can rename RDR groups, by using the menu entry Tools | Project Tools | Rename RDR groups.
How can I set the name/folder/group/role/properties of several RDRs all at once?
There are several functions, which allow this (with different feature sets):
  • Tools | Project Tools | set RDR properties with a regular expression
  • Tools | Project Tools | set property in multiple RDRs
  • Tools | Project Tools | find/replace in raw data record name/folder
These functions can all be found in the main menu of QuickFit.
