The "Tools" menu of the main screen provides several tools that make working with QuickFit easier:

  1. Project Tools:
    1. Fix Filepaths: fixes wrong/dangling filepaths in the project
    2. find/replace in raw data record name/folder: This option allows to find/replace text in the name and folder properties (or just one of these) of all raw data records. One undo step is provided (see next tool), so if you do something wrong, you may undo it. If you acivate this tool, a dialog will pop open, that allows you to select a phrase to find and a new phrase to replace the first one with. The find operation may be case-sensitive.
    3. undo last find/replace: Undos the last find/replace in RDR names/filders (see above for details).
    4. set property in multiple RDRs:: This tool allows you to add/change a property in several raw data records at once. You may specify a property name, value and type. Then you choose the RDRs to alter and whether new properties should be created and whether existing ones should be altered.
