QuickFit 3

QuickFit 3 is in itself not a fitting application (as the name may suggest), but more or less just a data manager with a plugin interface. All data formats and evaluation routines (e.g. fitting ...) are implemented as plugins. So it is possible to extend QuickFit in many ways and still use a common set of tools (project management ...). It is even possible to write plugins that transform QuickFit into an instrument control program for your experiment.

More informations about QuickFit 3, current versions and additional plugins may be found on the internet:

http://www.dkfz.de/Macromol/quickfit3/index.html
