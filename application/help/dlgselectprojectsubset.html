Select Project Subset Dialog

The menu entry $$bqtt:File | Special Project groups | open project subset$$ opens only a user-selectable subset of items (RDRs and evaluations) from an existing QuickFit project. This may help, if you e.g. want to split a project that became too large to fit into memory.
