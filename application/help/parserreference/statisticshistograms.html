The boolean normalize can be set to true (default: false), which results in a normalized histogram (normalized with the sum of all values, used for the histogram). This function ignores all NAN or INF values in x! $$startbox_example$$ x=[0,2,3,2,2,4,1,3,2,2,0,2,3,3,3];

histogrambins(x, 4)
