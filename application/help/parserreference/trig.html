Trigonometric Functions

  • sin(x) - sine function :
    calculates the sine of the argument (argument in radians). The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • cos(x) - cosine function :
    calculates the cosine of the argument (argument in radians). The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • tan(x) - tangent function :
    calculates the tangent of the argument (argument in radians). The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • asin(x) - inverse sine function :
    calculates the inverse sine of the argument (result in radians). The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • acos(x) - inverse cosine function :
    calculates the inverse cosine of the argument (result in radians). The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • atan(x) - inverse tangent function :
    calculates the inverse tangent of the argument (result in radians). The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • atan2(x,y) - inverse tangent function of two arguments:
    calculates the inverse tangent of two arguments (result in radians) $$bmath:x > 0 : \mbox{atan2}(y, x) = \arctan\left(\frac{y}{x}\right)$$ $$bmath:x < 0, y \geq 0 : \mbox{atan2}(y, x) = \arctan\left(\frac{y}{x}\right) + \pi$$ $$bmath:y < 0 , x < 0 : \mbox{atan2}(y, x) = \arctan\left(\frac{y}{x}\right) - \pi$$ $$bmath:y > 0 , x = 0 : \mbox{atan2}(y, x) = +\frac{\pi}{2}$$ $$bmath:y < 0 , x = 0 : \mbox{atan2}(y, x) = -\frac{\pi}{2}$$ $$bmath:y = 0, x = 0 : \mbox{atan2}(y, x) = \text{undefined}$$ The arguments x and y may be vectors, then the return value is also a vector of numbers where the function has been applied to each pair of entries of the input vectors.
