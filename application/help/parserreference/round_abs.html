rounding and absolute values

  • abs(x) - absolute value of the argument :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • ceil(x) - next larger integer number :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • floor(x) - next lower integer number :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • trunc(x) - cuts the number to an integer :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • round(x) - rounds the number to an integer :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • roundsig(x, digits) - rounds the number to a number with the given number of significant digits :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
