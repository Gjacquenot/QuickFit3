random numbers

The math parser uses internally a Mersenne Twister random number generator.

$$funcref_start$$ rand() - returns a random number between 0 and 1 (equally distributed) : $$funcref_description$$ The function rand() accepts up to two arguments:
  • rand(N) returns a random number between 0 and N
  • rand(N1, N2) returns a random number between N1 and N2
$$funcref_end$$ $$funcref_start$$ randint() - returns an integer valued random number between 0 and 232-1 (equally distributed) : $$funcref_description$$ The function randint() accepts up to two arguments:
  • randint(N) returns a random number between 0 and N
  • randint(N1, N2) returns a random number between N1 and N2
$$funcref_end$$ $$funcref_start$$ randnorm() - returns an normally distributed random number with mean 0 and variante 1 : $$funcref_description$$ The function randnorm() accepts up to two arguments:
  • randnorm(VAR) returns a random number normally distributed with mean 0 and variance VAR
  • randnorm(MEAN, VAR) returns a random number normally distributed with mean MEAN and variance VAR
$$funcref_end$$ $$funcref_start$$ randbool() - returns random boolean vector with a propability of true of 0.5 (by default). The propability can also be given as a fucntion parameter. : $$funcref_description$$ The function randbool() accepts up to two arguments:
  • randbool(0.2) returns a boolean that is true with a propability of 0.2
  • randbool() returns a boolean that is true with a propability of 0.5
$$funcref_end$$ $$funcref_start$$ srand(x) - seed the internal random number generator : $$funcref_description$$ $$funcref_end$$
