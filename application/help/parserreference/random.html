random numbers

The math parser uses internally a Mersenne Twister random number generator.

  • rand() - returns a random number between 0 and 1 (equally distributed) :
    The function rand() accepts up to two arguments:
    • rand(N) returns a random number between 0 and N
    • rand(N1, N2) returns a random number between N1 and N2
  • randint() - returns an integer valued random number between 0 and 232-1 (equally distributed) :
    The function randint() accepts up to two arguments:
    • randint(N) returns a random number between 0 and N
    • randint(N1, N2) returns a random number between N1 and N2
  • randnorm() - returns an normally distributed random number with mean 0 and variante 1 :
    The function randnorm() accepts up to two arguments:
    • randnorm(VAR) returns a random number normally distributed with mean 0 and variance VAR
    • randnorm(MEAN, VAR) returns a random number normally distributed with mean MEAN and variance VAR
  • randbool() - returns random boolean vector with a propability of true of 0.5 (by default). The propability can also be given as a fucntion parameter. :
    The function randbool() accepts up to two arguments:
    • randbool(0.2) returns a boolean that is true with a propability of 0.2
    • randbool() returns a boolean that is true with a propability of 0.5
  • srand(x) - seed the internal random number generator :

    • random vectors

    The math parser uses internally a Mersenne Twister random number generator.

    • randvec(items) - returns a vector (number of elements given in items) of random number between 0 and 1 (equally distributed) :
      The function randvec() accepts up to three arguments:
      • randvec(items, N) returns a random number vector with items elements between 0 and N
      • randvec(items, N1, N2) returns a random number vector with items elements between N1 and N2
    • randvec_seed(items, seed) - returns a vector (number of elements given in items) of random number between 0 and 1 (equally distributed). The RNG is seeded by seed. :
      The function randvec_seed() accepts up to four arguments:
      • randvec_seed(items, N,seed) returns a random number vector with items elements between 0 and N
      • randvec_seed(items, N1, N2,seed) returns a random number vector with items elements between N1 and N2
    • randintvec(items) - returns a vector (number of elements given in items) of integer valued random number between 0 and 232-1 (equally distributed) :
      The function randintvec(items) accepts up to three arguments:
      • randintvec(items, N) returns a random number vector with items elements between 0 and N
      • randintvec(items, N1, N2) returns a random number vector with items elements between N1 and N2
    • randintvec_seed(items, seed) - returns a vector (number of elements given in items) of integer valued random number between 0 and 232-1 (equally distributed). The RNG is seeded by seed.:
      The function randintvec_seed(items) accepts up to three arguments:
      • randintvec_seed(items, N, seed) returns a random number vector with items elements between 0 and N
      • randintvec_seed(items, N1, N2, seed) returns a random number vector with items elements between N1 and N2
    • randnormvec(items) - returns a vector (number of elements given in items) of normally distributed random number with mean 0 and variante 1 :
      The function randnormvec() accepts up to three arguments:
      • randnormvec(items, VAR) returns a random number vector with items elements normally distributed with mean 0 and variance VAR
      • randnormvec(items, MEAN, VAR) returns a random number vector with items elements normally distributed with mean MEAN and variance VAR
    • randnormvec_seed(items, seed) - returns a vector (number of elements given in items) of normally distributed random number with mean 0 and variante 1. The RNG is seeded by seed.:
      The function randnormvec_seed() accepts up to four arguments:
      • randnormvec_seed(items, VAR, seed) returns a random number vector with items elements normally distributed with mean 0 and variance VAR
      • randnormvec_seed(items, MEAN, VAR, seed) returns a random number vector with items elements normally distributed with mean MEAN and variance VAR
    • randboolvec(items) - returns a boolean vector (number of elements given in items) with elememts that are randomly choosen. the propability for true is 0.5 by default, but can also be given as a second parameter:
      The function randboolvec() accepts up to two arguments:
      • randboolvec(10, 0.2) returns a boolean vector with 10 elements and a propability of true of 0.2
      • randboolvec(10) returns a boolean vector with 10 elements and a propability of true of 0.5
    • randboolvec_seed(items, seed) - returns a boolean vector (number of elements given in items) with elememts that are randomly choosen. The propability for true is 0.5 by default, but can also be given as a second parameter. The RNG is seeded by seed:
      The function randboolvec_seed() accepts up to three arguments:
      • randboolvec_seed(10, 0.2, seed) returns a boolean vector with 10 elements and a propability of true of 0.2
      • randboolvec_seed(10, seed) returns a boolean vector with 10 elements and a propability of true of 0.5
