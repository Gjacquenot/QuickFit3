random numbers

The math parser uses internally a Mersenne Twister random number generator.

  • rand() - returns a random number between 0 and 1 (equally distributed) :
    The function rand() accepts up to two arguments:
    • rand(N) returns a random number between 0 and N
    • rand(N1, N2) returns a random number between N1 and N2
  • randint() - returns an integer valued random number between 0 and 232-1 (equally distributed) :
    The function randint() accepts up to two arguments:
    • randint(N) returns a random number between 0 and N
    • randint(N1, N2) returns a random number between N1 and N2
  • randnorm() - returns an normally distributed random number with mean 0 and variante 1 :
    The function randnorm() accepts up to two arguments:
    • randnorm(VAR) returns a random number normally distributed with mean 0 and variance VAR
    • randnorm(MEAN, VAR) returns a random number normally distributed with mean MEAN and variance VAR
