Vector Manipulation Functions

  • sort(x) - sorts the vector x in ascending order:
  • dsort(x) - sorts the vector x in descending order :
  • shuffle(x) - shuffle the vector x randomly :
  • reverse(x) - reverse the vector x :
    example: [1,2,3] -> [3,2,1]
  • concat(x1, x2, ...) - concatenate the parameter vectors x1, x2, ... :
    example: concat([1,2,3], [4,5,6]) -> [1,2,3,4,5,6]
  • removeall(x, val) - remove all occurenced of value val from vector x :
  • filterfinite(x) - remove all infinte/nan numbers from number vector x :
  • filternumok(x) - remove all infinte/nan numbers from number vector x :
  • remove(x, index) - remove the elements indexed by index from vector x :
  • contains(x, v) - returns true if the element v is contained in the vector x. If both elements are string, it returns true, if the substring v appears in x. :
  • containssubstring(x, v) - returns true if the string v is contained in the string x. If x is a string vector, the function returns a vector of booleans. If v and x are string vector, the index of the first elemnt in v is returned, which is contained in each element ofx. :
  • find(x, v), find(x) - returns a list with the array indices of all occurences of v in the vector x. If v is omitted, the functions expects a boolean vector as x and finds all occurences of true :
    This function can be used to select items from an array with a logic expression, e.g. find(x==4) will return the index list of all items in x that have the value 4. The result can be used in an array access, i.e. x[find(x>4)] will return a vector containing only those elements in x where the value is larger than 4. $$see:    select(), findfirst()$$
  • findfirst(x, v), findfirst(x) - returns the first array index of an occurences of v in the vector x. If v is omitted, the functions expects a boolean vector as x and finds the first occurence of true :
    If both arguments are vectors, the occurence of every element in x is searched in v. If an elemnt is not found, the index -1 is returned.
  • select(x, cond) - returns only those elements of a vector x where the boolean vector cond contains true. Both vectors have to have the same number of elements. :
    This function can be used to filter vectors according to a boolean expression, e.g. select(x, x>4) will return a vector containing only those elements in x where the value is larger than 4. This can also be implemented using x[    find(x>4)]. So this function is a more expressive short-form for that. $$see:findfirst(), find()$$
  • length(x) - returns the length of vector x :
  • dot(x,y) - returns the dot product of two numeric vectors x and x :
  • alltrue(x) - returns true if all elements of the boolean vector x are true :
  • allfalse(x) - returns true if all elements of the boolean vector x are false :
  • anytrue(x) - returns true if any element of the boolean vector x is true :
  • anyfalse(x) - returns true if any element of the boolean vector x is false :
  • countoccurences(x, value) - returns the number of elements in the vector x that have value value :
  • unique(x) - returns a list containing all elements in x, if any value appears more than once in x it is returned only once. The order is the order in the initial data vector x.
    Example:
    x=[1,3,2,1,2,3,3,2,1]; unique[x] -> [1,3,2]
  • first(x) - returns the first element in the vector x:
  • last(x) - returns the last element in the vector x:
  • item(x, idx) - If idxis an integer vector, this function returns the elements of vector x referenced by integer vector idx. If idxis vector of boolean, this function returns the elements of vector x where the according element in idx is true. The argument x may be a vector of numbers, boolean or strings, or a string. In the latter case the function accesses characters in the string.
    $$example:item([1,2,3,4,5,6,7,8,9], [1,3,5]) will return [2,4,6]$$ $$example:item([1,2,3,4,5,6,7,8,9], [true,true,true,false,false,false,true,false,true]) will return [1,2,3,7,9]$$ $$example:item("abcdefghij", [1,3,5]) will return "bdf"$$ $$example:item("abcdefghi", [true,true,true,false,false,false,true,false,true]) will return "abcgi"$$
  • runningaverage(x, width) - calculates the running average over vector x with width width. The argument x has to be a vector of numbers and width an integer >0
    The functions averages width consecutive entries in x, if less than width entries remain in x, they are omitted. $$example:runningaverage([1,2,3,4,5,6,7,8,9], 3) will return [2,5,8]$$
