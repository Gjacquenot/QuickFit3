Vector Manipulation Functions

  • sort(x) - sorts the vector x in ascending order:
  • dsort(x) - sorts the vector x in descending order :
  • shuffle(x) - shuffle the vector x randomly :
  • reverse(x) - reverse the vector x :
    example: [1,2,3] -> [3,2,1]
  • concat(x1, x2, ...) - concatenate the parameter vectors x1, x2, ... :
    example: concat([1,2,3], [4,5,6]) -> [1,2,3,4,5,6]
  • removeall(x, val) - remove all occurenced of value val from vector x :
  • remove(x, index) - remove the elements indexed by index from vector x :
  • find(x, v), find(x) - returns a list with the array indices of all occurences of v in the vector x. If v is omitted, the functions expects a boolean vector as x and finds all occurences of true :
    This function can be used to select items from an array with a logic expression, e.g. find(x==4) will return the index list of all items in x that have the value 4. The result can be used in an array access, i.e. x[find(x>4)] will return a vector containing only those elements in x where the value is larger than 4 $$see:    select()$$
  • select(x, cond) - returns only those elements of a vector x where the boolean vector cond contains true. Both vectors have to have the same number of elements. :
    This function can be used to filter vectors according to a boolean expression, e.g. select(x, x>4) will return a vector containing only those elements in x where the value is larger than 4. This can also be implemented using x[    find(x>4)]. So this function is a more expressive short-form for that,. $$see:find()$$
  • length(x) - returns the length of vector x :
  • dot(x,y) - returns the dot product of two numeric vectors x and x :
  • alltrue(x) - returns true if all elements of the boolean vector x are true :
  • allfalse(x) - returns true if all elements of the boolean vector x are false :
  • contains(x, value) - returns true if the vector x contains at least one element of value value :
  • countoccurences(x, value) - returns the number of elements in the vector x that have value value :
