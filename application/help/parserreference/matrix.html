Number Matrices

$$funcref_start$$ doublematrix() / doublematrix(rows) / doublematrix(rows,columns) / doublematrix(rows,columns,value) - creates a matrix of floating-point numbers: $$funcref_description$$
  • If no parameters are given, the resulting matrix is empty
  • If one parameter rows is given, the resulting matrix has a size rows*rows and each entry contains 0.
  • If two parameters rows,columns are given, the resulting matrix has a size rows*columns and each entry contains 0.
  • If three parameter rows,columns,value are given, the resulting matrix has a size rows*rows and each entry contains value.
$$funcref_end$$

Boolean Matrices

$$funcref_start$$ boolmatrix() / boolmatrix(rows) / boolmatrix(rows,columns) / boolmatrix(rows,columns,value) - creates a matrix of booleans: $$funcref_description$$
  • If no parameters are given, the resulting matrix is empty
  • If one parameter rows is given, the resulting matrix has a size rows*rows and each entry contains false.
  • If two parameters rows,columns are given, the resulting matrix has a size rows*columns and each entry contains false.
  • If three parameter rows,columns,value are given, the resulting matrix has a size rows*rows and each entry contains value.
$$funcref_end$$

Matrix reshaping

$$funcref_start$$ reshape(x, rows, cols) - creates a matrix with all entries (row-major order) from the given matrix x, but with a new size of rows * cols entries: $$funcref_description$$ If x does not have enough entries to fill the new matrix, an error will be generated. $$funcref_end$$
