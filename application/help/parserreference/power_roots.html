Powers and Roots

  • sqrt(x) - square root :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • cbrt(x) - sube root :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
  • sqr(x) - square of the argument (power 2) :
    The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector.
