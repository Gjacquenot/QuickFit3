$$qf_commondoc_header.start$$ $$qf_commondoc_header.end$$ $$funcref_start$$ fitfunction_fit(fitalg, ffid, initparams, fix, x, y, w) - fits the supplied fit function to the data vectors vector x,y : $$funcref_description$$ This function solves the least squares fit problem: $$bmath:\vec{p}^\ast=\argmin_{\vec{p}}\sum_i\left(\frac{y_i-f(x_i,\vec{p})}{w_i}\right)$$ where $$math:f(x,\vec{p})$$ is the fit model specified by the fit function ID ffid and the fit algorithm fitalg is used. The parameter initparams contains the starting values and fix (optional!) determines, which parameters are fixed (boolean vector!). If fix is missing, the initial fix-settings of the specified fit function are used. The vector of weights w is also optional and $$math:w_i=1$$ is used, if it is missing. $$funcref_end$$
