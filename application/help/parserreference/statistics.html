Statistics Functions

  • mean(x) - returns the mean of vector x :
    This function ignores all NAN or INF values in x!
  • sum(x) - returns the sum of all elements in vector x :
    This function ignores all NAN or INF values in x! See also     the extended sum function!
  • sum2(x) - returns the sum of the squares of all elements in vector x :
    This function ignores all NAN or INF values in x! See also     the extended sum function!
  • prod(x) - returns the product of all elements in vector x :
    This function ignores all NAN or INF values in x! See also     the extended sum function!
  • std(x) - returns the standard deviation of vector x :
    This function ignores all NAN or INF values in x!
  • var(x) - returns the variance of vector x :
    This function ignores all NAN or INF values in x!
  • min(x) - returns the minimum of vector x :
    This function ignores all NAN or INF values in x!
  • max(x) - returns the maximum of vector x :
    This function ignores all NAN or INF values in x!
  • median(x) - returns the median of vector x :
    This function ignores all NAN or INF values in x!
  • quantile(x, p) - returns the p-quantile of vector x, $$math:0\leq p\leq1$$ :
    This function ignores all NAN or INF values in x!
