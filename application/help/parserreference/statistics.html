Statistics Functions

  • mean(x) - returns the mean of vector x :
    This function ignores all NAN or INF values in x!
  • sum(x) - returns the sum of all elements in vector x :
    This function ignores all NAN or INF values in x! See also     the extended sum function!
  • sum2(x) - returns the sum of the squares of all elements in vector x :
    This function ignores all NAN or INF values in x! See also     the extended sum function!
  • prod(x) - returns the product of all elements in vector x :
    This function ignores all NAN or INF values in x! See also     the extended sum function!
  • std(x) - returns the standard deviation of vector x :
    This function ignores all NAN or INF values in x!
  • var(x) - returns the variance of vector x :
    This function ignores all NAN or INF values in x!
  • skewness(x) - returns the skewness of vector x :
    This function ignores all NAN or INF values in x!
  • moment(x, p) - returns the central moment of order p of vector x:
    This function ignores all NAN or INF values in x!
  • ncmoment(x, p) - returns the non-central moment of order p of vector x:
    This function ignores all NAN or INF values in x!
  • corrcoeff(x, y) - returns the correlation coefficient (Pearson) of vectors x and y:
    This function ignores all NAN or INF values in x!
  • min(x) - returns the minimum of vector x :
    This function ignores all NAN or INF values in x!
  • max(x) - returns the maximum of vector x :
    This function ignores all NAN or INF values in x!
  • median(x) - returns the median of vector x :
    This function ignores all NAN or INF values in x!
  • quantile(x, p) - returns the p-quantile of vector x, $$math:0\leq p\leq1$$ :
    This function ignores all NAN or INF values in x!
  • mad(x) - returns the median absolute deviation about the median (MAD), a robust measure of sample deviation of vector x :
    $$bmath:\mbox{MAD}(\vec{x})=\mbox{Med}\left\{|\vec{x}-\mbox{Med}(\vec{x})|\right\}$$ This function ignores all NAN or INF values in x! $$see:Ricardo A. Maronna, R. Douglas Martin, Victor J. Yohai: "Robust Statistics: Theory and Methods", Wiley, 2006, ISBN: 978-0-470-01092-1$$
  • nmad(x) - returns the normalized median absolute deviation about the median (NMAD), a robust measure of sample deviation of vector x :
    $$bmath:\mbox{NMAD}(\vec{x})=\frac{\mbox{MAD}(\vec{x})}{0.6745}=\frac{\mbox{Med}\left\{|\vec{x}-\mbox{Med}(\vec{x})|\right\}}{0.6745}$$ This function ignores all NAN or INF values in x! $$see:Ricardo A. Maronna, R. Douglas Martin, Victor J. Yohai: "Robust Statistics: Theory and Methods", Wiley, 2006, ISBN: 978-0-470-01092-1$$

Indexed Statistics Functions

  • indexedmean(x, idx) - returns a vector of the averages of values in x, that have the same index in idx:
    The order is the order of first appearance of each index in idx and the same order, as returned by     unique(idx).
    Example:
    x=[1,2,3,4,5,6,7,8,9];
    i=[1,2,3,1,2,3,1,2,3];
    indexedmean(x,i) -> [4,5,6]
  • indexedstd(x, idx) - returns a vector of the standard deviations of values in x, that have the same index in idx:
    The order is the order of first appearance of each index in idx and the same order, as returned by     unique(idx).
