Distribution Functions

$$funcref_start$$ gauss(x, sigma) - normalized gaussian function : $$funcref_description$$ $$bmath:\mbox{gauss}(x,\sigma)=\frac{1}{\sigma\cdot\sqrt{2\pi}}\cdot\exp\left(-\frac{1}{2}\cdot\frac{x^2}{\sigma^2}\right)$$ The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector. $$funcref_end$$ $$funcref_start$$ gaussnn(x, sigma) - non-normalized gaussian function : $$funcref_description$$ $$bmath:\mbox{gaussnn}(x,\sigma)=\exp\left(-\frac{1}{2}\cdot\frac{x^2}{\sigma^2}\right)$$ The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector. $$funcref_end$$
