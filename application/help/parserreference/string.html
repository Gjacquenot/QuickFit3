Regular Expressions Functions

This chapter groups QF math parser functions for regular expressions. Please see the QuickFit regular expressions help page for details on the supported set of regular expressions and their syntax.

  • regexpcap(regexp, strings, cap_id=1, default_string="") - applies the given regular expression regexp to the given string or string vector in strings. The function returns the cap_id-th captured substring (the default is cap_id=1). This function uses non-minimal matching and is non-case-sensitive.:
    If the regular expression does not match, the given default_string is returned.
  • regexpcontains(regexp, strings) - returns true, if the given regular expression regexp is present in the given string or string vector (item-wise for string vectors, i.e. returns a boolean vector) in strings. This function uses non-minimal matching and is non-case-sensitive.:
  • regexpindexin(regexp, strings) - returns index of the first occurence (or -1), of the given regular expression regexp in the given string or string vector (item-wise for string vectors, i.e. returns a number vector) in strings. This function uses non-minimal matching and is non-case-sensitive.:
