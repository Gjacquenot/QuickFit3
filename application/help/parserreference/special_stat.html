Discrete Propability Density Functions (PDFs)

$$funcref_start$$ poissonpdf(k, lambda) - poisson propability density function $(P_\lambda(k))$ for an integer $(k\geq0)$ and the parameter $(\lambda)$: $$funcref_description$$ $$bmath:P_\lambda(k)=\frac{\lambda^k}{k!}\cdot\mathrm{e}^{-\lambda}$$ The argument k may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector, all with the same parameter $(\lambda)$. $$funcref_end$$ $$funcref_start$$ binomialpdf(k, N, p) - binomial propability density function $(B(k|N,p))$ for an integer $(k\geq0)$, an experiment size $(N)$ and a propability $(0\leq p\leq1)$: $$funcref_description$$ $$bmath:B(k|N,p)=\left(\stackrel{N}{k}\right)\cdotp^k\cdot(1-p)^{N-k}$$ The argument k may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector, all with the same parameters $(N,p)$. $$funcref_end$$

Continuous Propability Density Functions (PDFs)

$$funcref_start$$ gauss(x, sigma) - normalized gaussian function : $$funcref_description$$ $$bmath:\mbox{gauss}(x,\sigma)=\frac{1}{\sigma\cdot\sqrt{2\pi}}\cdot\exp\left(-\frac{1}{2}\cdot\frac{x^2}{\sigma^2}\right)$$ The argument x may be a vector, then the return value is also a vector of numbers where the function has been applied to each entry of the input vector. $$funcref_end$$
