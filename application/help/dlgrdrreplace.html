Replace RDR Property Dialog

This dialog allows to replace text in certain properties of a raw data record with another text. It can be opened by clicking $$bqtt:Tools | project tools | find/replace in raw data record name/folder$$ Please enter the search string and the text to replace it with and select the properties that should be searched. The function will then change all occurences without further user interaction. It is possible though to undo thre last replacement with a click on $$bqtt:Tools | project tools | undo last find/replace$$ $$see:A more advanced function is replacing with regular expressions$$
