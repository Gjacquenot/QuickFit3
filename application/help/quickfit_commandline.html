QuickFit can be run with some command line options, which allow to switch on/off special features:

  • --nosplash: does not display the splash screen
  • --getbits: outputs whether the application is 32- or 64-bit on the standard output and exits immediately
  • --verboseplugin: prints additional information when searching and loading QuickFit plugins to the main log pane. This is useful, when a plugin is not loaded (e.g. because a driver DLL is not found or the versions do not match the installed QUickFit library version)
  • The last argument may be a .qfp QuickFit project, which is loaded on startup. So QuickFit may be associated with .qfp-files on your operating system
