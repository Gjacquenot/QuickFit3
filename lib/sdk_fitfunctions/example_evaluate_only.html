This is an example of a fit function from a shared library!!!

It implements a simple sine function: $$bmath:f(x,\vec{p})=p_0+p_1\cdot\sin(2\pi\cdot p_2\cdot x)$$ where the first parameter $$math:p_0$$ is an offset, $$math:p_1$$ is the amplitude and $$math:p_2$$ is the frequency of oscillations.
